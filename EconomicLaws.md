## 第一章 法律基本原理
- 法律效力：宪法 → 法律 → 行政法规 → A.部门规章 B.地方性法规 → 同级政府规章
- 法律事实：
  1. 事件：人的出生与死亡；自然灾害与意外事件；时间的经过。
  2. 人的行为：
    + 法律行为：意思表示为要素，要约承诺、签发票据等。
    + 事实行为：无需意思表示要素，发现埋藏物、拾得遗失物、先占、加工、著作、无因管理、侵权行为、建造拆除等。*背诵*

## 第二章 基本民事法律制度
诉讼时效中断、表见代理

### 第一节 民事法律行为制度
- 单方法律行为：
  1. 授权和撤销委托
  2. 免除债务
  3. 追认无权代理
  4. 订立遗嘱
  5. 出具担保函
- 赠与是多方法律行为！
- 民事行为有效性分类：

|行为|合同|其他单方民事行为|
|---|---|---|
|限制行为能力人不能独立实施的民事行为|效力待定|无效|
|欺诈、胁迫|1) 不损害国家利益：可变更可撤销(仅受害方有撤销权)<br> 2) 损害国家利益：无效|无效|
|乘人之危(客观存在的危难+故意+严重损害)|可变更可撤销(**无论是否损害国家利益**，仅受害方有撤销权)|无效|
|恶意串通损害他人利益(是违法的)|无效|无效|
|违反法律或社会公共利益(指违反法律+行政法规的强制性规范)|无效|无效|
|以合法形式掩盖非法目的|无效|无效|
|重大误解+显失公平(订立时)|可变更可撤销<br> (双方都有撤销权，过错方撤销后赔偿)|可变更可撤销|

- 可变更可撤销合同的撤销权是形成权，知道或应当知道后 **1年** 的除斥期间经过后权利本体消灭；人民法院不会主动释明撤销权的存在。撤销后双方返还的请求权诉讼时效从被撤销之日起算。

### 第二节 代理制度
- 委托书授权不明，被代理人和代理人连带责任
- 代理权的滥用（有代理权，不好好用）：
  1. 自己代理 → 无权代理
  2. 双方代理 → 无权代理
  3. 代理人和第三人恶意串通 → 无效
- 狭义的无权代理（没有代理权、超越代理权、代理权终止后）
  1. 订立的合同效力待定
  2. 本人的追认（相对人不论善恶意均可催告 + **1个月内** + 默示拒绝），可用实际履行表示追认，追认权是形成权
  3. 善意相对人在本人追认前可以行使撤销权，也是形成权
- **表见代理**
  1. 构成要件：
    + 代理人无代理权
    + 相对人主观善意无过失
    + 客观上有可信赖的情形
    + 成立民事行为
  2. 效果：
    + 对外 = 有权代理
    + 相对人可以主张为无权代理行使撤销权，或者主张表见代理使得行为有效，本制度保护善意相对人利益。
- 不属于代理的情况：
  1. 行纪、寄售：行纪人和寄售中的代销人均是以自己的名义实施法律行为
  2. 代表：法定代表人和被代表的法人是同一民事主体
  3. 居间：就是通常说的中介，进行消息传递等，无独立意思表示
  4. 代人保管物品：不涉及第三人

### 第三节 诉讼时效制度
- 适用于 **债权请求权**，期间届满胜诉权消灭，不丧失起诉权和实体权利，人民法院不主动释明。
- 不适用的债权请求权：*背诵*
  1. 支付存款本金和利息
  2. 国债、金融债券和公开发行的企业债券兑付本息
  3. 基于投资关系的缴付出资请求权
- 种类(x4)与起算：
  1. 普通：2年
  2. 短期：1年 *背诵*
    + 身体受到伤害要求赔偿的
    + 出售质量不合格的商品未声明的
    + 延付或拒付租金的
    + 寄存财物被丢失或者毁损的
  3. 长期：4年
    + 涉外货物买卖合同争议
    + 技术进出口合同争议
  4. 最长：20年，从权利被侵害时起算，适用延长，但不适用中断中止
  5. 前三种起算：从知道或者应当知道其权利被侵害时起算
- 诉讼时效中止(暂停 pause)：因为无法行使所以暂停，最后 **6个月内** 可以中止。
  1. 不可抗力
  2. 其他障碍
- 诉讼时效中断(清零 clear)：积极行使了权利，所以时效重新计算。
  1. 提起诉讼，及其同等效力事项
  2. 当事人一方提出请求，其中对部分债权主张权利(比如行使抵押权)，中断效力及于剩余债权
  3. 义务人同意履行义务
    + 中断效力及于连带债权债务人
    + 提起代位权诉讼的，两个债权债务关系 **均** 中断
    + 债权转让的，通知到达债务人时中断
    + 债务承担的，债务承担意思表示到达债权人时中断

## 第三章 物权法律制度
with 合同法；change 浮动抵押

### 第一节 物权法律制度概述
- 主物与从物、原物与孳息物(是两个独立的物，没生出来的不算；果实、幼崽、利息、租金、中奖的奖金等)
- 物权法基本原则：
  1. 物权法定，其中法仅指全国人大和常委会制定的法律
  2. 客体特定
  3. 物权公示

### 第二节 物权变动
- 非基于法律行为的物权变动：再次处分时要先登记到自己名下，然后再变更登记到受让人名下
  1. 基于事实行为：建造、拆除，成就时发生效力
  2. 基于法律规定：继承、受遗赠，开始时发生效力
  3. 基于公法行为：法律文书、征收决定，生效时发生效力
- 无权处分(**合同是合同，物权是物权**)：注意与无权代理相区分，看以谁的名义
  1. 债权有效
  2. 物权效力待定
- 公式方式：
  1. 动产：一般交付生效，特殊动产（船舶、航空器、机动车）交付生效 + 登记对抗；现金最特别
    + 现实交付
    + 交付替代
      1. 简易交付，法律行为生效（如合同成立时）时成就
      2. 指示交付，指示到达实际占有人时成就（交付的其实是返还原物请求权，故如果指示不当，实际占有人可拒绝，则指示交付不成就）
      3. 占有改定，约定生效时成就
  2. 不动产：一般登记生效（房屋、建设用地使用权、抵押等），特殊（**土地承包经营权、地役权**）登记对抗
- 更正登记和异议登记：先更正登记，不同意再异议登记，异议登记后 **15日内** 不起诉，异议登记失效
- 预告登记：债权消灭或者能够进行不动产登记之日起 **三个月内** 不申请正式登记的，预告登记失效
  1. 预购商品房
  2. 以预购商品房设定抵押
  3. 房屋所有权转让和抵押
- 登记的地位，轻一 p.59-60，表3-2

### 第三节 所有权
- 共有：**一切约定优先**。有家庭关系以外，作按份共有推定；对外债权债务连带
  1. 共同共有
    + 家庭关系
    + 共同享有和承担债权债务
    + 重大修缮、处分时全体一致同意；管理费用共同负担
  2. 按份共有
    + 份额可以约定，否则按出资比例，最后等额
    + 重大修缮、处分时 **2/3以上份额** 同意；管理费用按份额负担
    + 转让份额时其他共有人有优先购买权
- **善意取得制度**
  1. 构成要件
    + 出让人合法占有但是无权处分（即委托物；遗失物诉讼时效知道后2年，若经转让取得，对价问题）
    + 受让人善意无过错（指受让之时）
    + 依法律行为转让（在交易中）
    + 合理对价（**不要求已经付款**）
    + 所有手续都办完了（交付、登记等）
  2. 法律效果
    + 受让人取得完整的所有权（原始取得）
    + 损害赔偿
- 动产所有权特殊取得
  1. 先占
  2. 拾得遗失物、发现埋藏物
    + 发招领公告 **6个月后** 无人认领，归国家所有，从而原所有人丧失所有权
    + 有费用偿还请求权；有悬赏广告时还有报酬请求权；拒不返还按侵权处理，两个请求权都没了
  3. 添附：看相对价值，有明显高下的归属较高一方，无明显高下的则共有；赔偿失去所有权之人
    + 附合
    + 混合
    + 加工

### 第四节 用益物权
- 建设用地使用权的取得，登记生效
  1. 创设取得
    + 无偿划拨再出让的，缴纳土地使用权出让金
    + 就拍卖所得价款，土地使用权出让金优先于抵押权受偿
  2. 移转取得
    + 转让建设中房地产，完成投资总额25%以上
- 建设用地使用权的期限 *背诵*
  1. 无偿划拨，没有期限限制
  2. 有偿出让按用途：
    + 居住用地，70年
    + 工业用地，50年
    + 教科文卫+体育，50年
    + 商业旅游娱乐用地，40年
    + 综合及其他用地，50年

### 第五节 担保物权
- **抵押权**：
  1. 不可分性
  2. 不承担未经同意转让的债务之担保责任
  3. 土地承包经营权仅许可公开方式取得的荒地
  4. 动产的浮动抵押 抵押物未定额
    + 主体限制 抵押物限制：流水线和产物
    + 财产确定的步骤
    + 登记了也不能对抗正常经营活动中 **已支付** 合理价款并取得抵押财产的买受人
  5. 房地一体原则
    + 城市房地产和建设用地使用权一起，但地带房的新增建筑物不算
    + 乡村不能地带房
  6. **登记** 合同生效不需要登记，物权变动要登记
    + 登记生效：不动产系列
    + 登记对抗：浮动抵押和可登记的动产，合同生效设立登记对抗，不可登记的动产实务上不做抵押物
  7. 担保范围：
    + 主债权&利息
    + 违约金&损害赔偿金
    + 保管&拍卖费用
  8. 物上代位：保险赔偿补偿金
  9. 禁止流押合同（约定不还钱抵押物所有权归债权人）
  10. 转让：区分动产/不动产
    + 须经抵押权人同意，否则无效（不动产肯定无效，动产有未登记+无权处分+善意取得情况）
    + 价款提前清偿或提存
    + 受让人可代为清偿债务并获得抵押物所有权（**即涤除权**）
  11. 到期不还钱后扣押抵押物，**此后** 抵押权人有孳息收取权，**不取得所有权**，而是加入抵押财产
  12. 后抵押不破先租赁（先抵押的，应书面告知承租人，否则损失赔偿；抵押权实现后租赁合同对受让人无约束力）；同时只有 **房屋租赁** 承租人才有优先购买权
  13. 实现抵押权：
    + 费用 → 利息 → 主债权
    + 顺位问题：
      1. 是否登记（仅登记对抗，登记生效的不登记就回家吧），都没登记 **按债权比例**
      2. 登记时间，一样的话按比例
      3. 在后的先到期，变卖价款超出在先债权部分受偿
      4. 在先的先到期，有剩余则提存，无剩余在后抵押权消灭
      5. 顺位金额都可协议变更，但未经其他债权人 **书面同意** 不得损害其利益，否则构成撤销事由（知道后 **1年内**）
      6. 自己放弃，其他担保人在放弃范围内免责
  14. 最高额抵押（债务未定额，ref:高杉Legal）
    + 一定期间内连续发生的债权
    + 抵押权不可部分转让
    + 债权确定
  15. 权利消灭
    + 债权消灭
    + 抵押权实现
    + 抵押物灭失，有赔钱则物上代位，抵押权不消灭
    + 混同：抵押权人获得抵押物所有权
  16. 抵押物部分灭失，残存部分仍担保债权之全部
- 质权：合同是合同，物权是物权。要书面合同
  1. 动产(交付设立) + 权利(有权利凭证交付，无权利凭证登记，票据、证券、知识产权、应收账款；**人身权不得出质**，名誉权肖像权、配偶权荣誉权等)
  2. 孳息收取权：有权收取 + 无所有权 + 加入质押财产
  3. 其他参照抵押权
- 留置权
  1. 若债务人为无权处分，债权人 **仍可以** 依善意取得制度行使留置权
  2. 法定，可以特约排除
  3. 成立要件：
    + 合法占有
    + 债权已到期
    + 占有和债权属同一法律关系（不限于合同），**企业之间不受此限**
  4. 其他参照质权
- 三种担保物权效力等级：
  1. 留置权人最优
  2. 登记的抵押权 > 质权 > 未登记的抵押权

## 第四章 合同法律制度
with 物权法、票据法、破产法；new 保证人的诉讼问题、民间借贷合同

### 第一节 合同的基本理论
- 诺成合同（大部分；赠与合同、质押合同）与实践合同（保管合同、自然人之间的借贷合同、定金合同，不给付为缔约过失而不是违约）
- 合同的相对性 债的相对性
  1. 主体
  2. 内容
  3. 责任（因第三人原因不能履约，仍应承担违约责任）
  4. 例外
    + 债的保全（代位权、撤销权）
    + 所有权让与不破租赁
    + 承包分包、分区段承运

### 第二节 合同的订立
- 要约&承诺
  1. 要约
    + 内容具体确定：at least 名称、标的、数量
    + 对方承诺后即受约束
    + 要约邀请（招标公告、招股说明书、商业广告），符合要约条件（某些广告、悬赏、商品房的具体允诺）视为要约
    + 到达生效
      1. 未生效时撤回任意，之前或同时到达即可
      2. 生效后撤销有限制：必须对方 **发出** 承诺通知前 **到达**，以及不得撤销情形(x3)：
        - 确定了承诺期限
        - 明示要约不可撤销
        - 对方有理由认为要约不可撤销，并已经为履行合同做了准备
    + 失效情形(x4)：
      1. 拒绝通知到达要约人
      2. 撤销要约
      3. 承诺期限届满未承诺
      4. 对方做出实质性变更（成为新要约，标的、数量、质量、价款、履行期限地点方式、违约责任、争议解决）
  2. 承诺
    + 承诺通知到达生效，生效时合同成立
    + 可以撤回，之前或同时到达
    + 不存在撤销的问题
    + 迟延的承诺（晚发了），除要约人及时承认外为新要约
    + 迟到的承诺（按时发，但是受阻了），除要约人及时拒绝外为有效承诺
    + 实质性变更，新要约；非实质性变更，除及时反对或已规定不得变更外，承诺有效并 **以承诺为准**
- 合同成立的时间地点
  1. 时间：
    + 承诺生效
    + 最后一个签名的时候
    + 一方履行了主要义务对方接受时（实际履行原则）
  2. 地点：
    + 生效地
    + 合同约定地
    + 最后一个签名地
- 格式条款：免除自己责任、加重对方责任、排除对方主要权利的条款无效；歧义按不利于提供者解释；有冲突时采用非格式条款
- 免责条款：对 造成对方人身伤害（不要求故意重大过失）、故意or重大过失造成对方财产损失的 免责条款无效
- 缔约过失责任：合同未成立/未生效/无效，信赖利益损失的赔偿

### 第三节 合同的效力
- 效力待定的合同
  1. 限制行为能力人不能独立订立的合同
  2. 无权代理人订立的合同
  3. 无处分权人订立的合同 **有效**

### 第四节 合同的履行
- 约定不明时
  1. 协议补充
  2. 按合同有关条款or交易习惯
  3. 按一些规则：
    + 质量：国家/行业/通常标准，合同目的标准
    + 价款：政府定价/指导价，履行地市场价
    + 履行地：接受货币方所在地、不动产所在地、履行义务方所在地
    + 履行期限：双方随时，允许必要准备时间
    + 履行方式：有利于实现合同目的的方式
    + 履行费用负担：由履行义务一方负担（比如交货环节的义务方即卖家负担运输费）
- 中止、提前、部分履行：借款人有提前履行的权利，其他情况债权人可拒绝
- 双务合同中的抗辩权 用于拒绝 **相应的** 履行要求
  1. 同时履行抗辩权
  2. 先履行抗辩权（后履行人用）
  3. 不安抗辩权（先履行人用）：确切证据；提供适当担保后继续；合理期限内未恢复也没担保，中止一方可以 **解除合同**
- **代位权**：针对债务人消极不行使债权，债权人为A，依次BC
  1. 行使条件：
    + 债权合法
    + AB的债权 **均到期**
    + 必须是金钱给付关系的债权
    + B的债权不专属于自身（基于抚养赡养继承等的给付请求权；劳动报酬、养老金、人身伤害赔偿请求权等）
  2. 管辖：C为被告，被告住所地管辖；被告承担诉讼费用，其他必要费用由债务人承担
  3. 法律效果：
    + 两个债权债务关系都消灭
    + C可向A主张对B的抗辩
    + AB、BC间的诉讼时效均中断
    + 相当于A就行使结果有优先受偿权（比如还有别的债权人）
  4. 以A自己的债权为限
  5. 必须依一定的诉讼程序进行，是合同保全制度中的一种手段
- **撤销权**：针对债务人积极侵害债权人的债权，不正当减少财产，撤销该处分
  1. 必须依一定的诉讼程序进行，是合同保全制度中的一种手段
  2. 成立要件：
    + A以自己的名义行使撤销权
    + A对B的债权有效，**到期不到期均可**
    + B不正当减少自己的财产
      1. 放弃债权（无论是否到期）、放弃担保或恶意延长履行期
      2. 无偿转让财产
      3. 明显不合理（与同类相比浮动超出30%）的低价卖高价买，并且 **相对人知情**
      4. B的处分行为有害于A债权的实现（B特别有钱就没关系）
  3. 行使范围以A的债权为限
  4. 行使的期限：
    + 知道后 **1年**
    + 发生后 **5年**，实体权利消灭（除斥期间，意思上像那个最长20年）
  5. 法律效果：
    + B的处分无效，双方返还
    + 恢复责任财产，A就行使结果 **并无优先受偿权**
  6. 管辖：B为被告，各种费用被告负担，第三人有过错的适当分担

### 第五节 合同的担保
- 保证、抵押、质押、留置、定金；约定和法定；人保、物保、金钱担保
- 反担保：留置和定金不可；债务人可抵押质押、其他人可抵押质押和保证
- 担保合同无效：
  1. 担保提供人主体资格不具备
  2. 以禁止流通、不可转让的财产设定担保
  3. 承担缔约过失责任：债权人A，债务人B，担保人C
    + 主合同有效：
      1. A无过错，BC连带
      2. AC有过错，先找B还钱，然后C的责任不超过B不还的 **1/2**
    + 主合同无效使得担保合同无效：
      1. C无过错，C不承担民事责任
      2. C有过错，先找B还钱，然后C的责任不超过B不还的 **1/3**
- **保证**：即人保
  1. 是AC之间的合同关系，是C的单务合同，出具担保函是单方法律行为。以下情况保证合同成立：
    + C在主合同上以 **保证人身份** 签章，不论主合同是否有保证条款
    + C出具担保函，A无异议接受
  2. 人数不限，不得作为保证人的有：主体资格不具备，保证合同 **无效**
    + 主债务人
    + 国家机关、公益目的组织
    + 法人的职能部门、法人的分支机构（法人书面授权分支机构的除外）
  3. 保证方式：
    + 一般保证和连带责任保证，推定是连带责任保证
      1. 一般保证人有先诉抗辩权（先对债务人强制执行）
      2. 先诉抗辩权不得行使情形(x3)：
        - 债务人住所变更难以找到
        - **债务人破产案件被受理**
        - 保证人书面放弃先诉抗辩权
    + 单独保证和共同保证（按份/连带，看 **与债权人的约定**，推定是连带）
      1. 连带共同保证是保证人之间的连带
      2. 连带共同保证的保证人内部可以约定份额，推定均分
      3. 一人承担责任后可向债务人追偿，之后可对内要求按份额承担
  4. 保证范围参照抵押
  5. 主合同各种变更：未经保证人/担保人书面同意，改变/加重的部分不承担保证责任
  6. 保证期间：除斥期间，过期权利消灭，债务到期之日起算
    + 未约定的：**6个月**
    + 约定的期间早于或者等于主债务履行期限的，视为没有约定
    + 约定不明：**2年**
    + 要求清偿破产程序中未受清偿部分的：自破产程序终结后 **6个月**
  7. 在保证期间中，债权人主张权利的，保证责任确定。诉讼时效2年：
    + 主张权利方式和起算：
      1. 连带：向保证人要求承担保证责任，自此起算
      2. 一般：对债务人提起诉讼/仲裁，自判决/裁决文书生效之日起算
    + 主债务时效中止和中断波及：*背诵*
      1. 主债务中断：连带 **不中断**，一般中断
      2. 主债务中止：连带和一般都中止
  8. 共同担保（有人保有物保）下责任顺序：
    + 看约定
    + 债务人自己提供的物保
    + 第三人提供的人保和物保（此顺位上若人保物保并存，人保/物保其中一方承担责任后不能向对方追偿，只能向债务人追偿，**此处不同于连带共同保证**）
  9. 追偿权：自承担责任之日起算2年
  10. 涉及保证人的诉讼：民间借贷债权人仅起诉一般保证人的，人民法院应当追加债务人为共同被告，其他情况下都是看心情追加。
- 定金：实践合同，实际交付定金时生效
  1. 交付即移转所有权，货币特点（“现金最特别”）
  2. 罚则：给定金的不履行不能要回，收定金的不履行 **双倍返还**（其实是恢复原状+再给一倍的定金，所以两边实质上是一样的）
  3. 约定的定金数额超过主合同标的额 **20%** 的无效（超出部分视为预付款）
  4. 定金罚则适用：
    + 不完全履行按比例
    + 使合同目的不能实现时才适用
    + 因不可抗力、意外事件不适用
    + 因第三人原因则适用（合同的相对性），向第三人追偿
  5. 同一合同中同时有定金和违约金条款，一方违约时对方 **不能要求同时适用**

### 第六节 合同的变更与转让
- 债权转让：
  1. 成立生效，通知债务人后对债务人发生效力
  2. 根据合同性质（特定身份，出版、赠与、委托等）、约定、法律规定有不得转让情形
  3. 从权利（抵押权等）一并转让
  4. 债务人对原债权人的抗辩可以对新债权人提出
- 债务承担：
  1. 需要取得债权人同意
  2. 债务承担协议在债权人同意时生效
- 债权债务概括移转：一般因为法人合并分立。除 **与债权人** 另有约定外，分立之后的主体债权债务 **连带**（仅有内部约定则不得向外对抗）

### 第七节 合同的终止
- 清偿：没钱全还的，偿还顺序未约定的，按照对债权人最有利的原则
  1. 费用 - 利息 - 主债务
  2. 到期早的、担保少的、负担大的、最后按比例
- 解除：合意/法定
  1. 合意解除：事先约定/事后协商
  2. 可以法定解除的情形：故意/合同目的不能实现/法定
    + 不可抗力使合同目的不能实现，双方均可（若是可替代物等灭失可替代履行，则不能解除）
    + 对方明确表示或行为不履行主要债务
    + 对方迟延履行主要债务，催告后合理期限内未履行
    + 对方有违约行为致使不能实现合同目的（此条重点在目的，不用主要或催告），比如无权处分
    + 一些法律规定事项：
      1. 不安抗辩权相关
      2. 定作人可以随时解除承揽合同
      3. 托运人可以单方解除货运合同
      4. 委托合同的双方均可
      5. 不定期租赁的双方均可
      6. 情势变更（无法预见、非不可抗力、非属商业风险，使得显失公平）
  3. 行使解除权应通知对方，到达生效；异议期无约定推定 **3个月**
  4. 解除后，终止履行/恢复原状/补救措施/赔偿损失，不影响结算/清理/争议解决条款的效力
- 抵销：法定/约定
  1. 法定抵销
    + 条件：
      1. 同等效力的互负债权债务
      2. 给付为同一种类（不要求数量一样）
      3. 均到期
      4. 均可抵销（故意侵权、提供劳务、约定等情形不可抵销）
    + 条件1or3不满足的，仅可能吃亏的一方可以主张抵销（即：诉讼时效未届满的债权人、债务未到期的债务人等）
    + 形成权，通知到达生效，不得附条件和期限
  2. 约定抵销：协商一致则随便，标的物种类品质不一样也没事
- 提存（以清偿为目的）
  1. 原因：债权人的情况使得难以履行债务
    + 无正当理由（质量不合格等）拒绝受领
    + 下落不明（此情形没有通知义务，做不到）
    + 死亡/丧失行为能力无继承人/监护人
  2. 法律效果：
    + 债消灭，等同于给了债权人
    + 债权人领取的权利时效 **5年**（不变期间），不行使则消灭，提存物归国家
- 免除与混同（债权债务同归于一人，不涉及第三人利益）

### 第八节 违约责任
- 以合同有效为前提，违反合同义务的责任，相对性
- 违约责任、侵权责任竞合：只能选一个提起诉讼请求
- 分不履行和不适当履行；违约责任赔偿采用填坑原则
- 责任承担方式(x3)：
  1. 继续履行
  2. 补救措施
  3. 损害赔偿
    + 赔偿损失：赔偿额应当相当于净损失
    + 支付违约金
      1. 约定的违约金和实际损失不匹配的，调低下限标准为1.3倍，调高上限标准为1倍
      2. 迟延履行的违约金支付后，仍应履行债务
    + 适用定金罚则
      1. 不可同时适用定金和违约金条款
      2. 若定金不足以弥补损失，可以并处赔偿，总额不超过违约损失即可
      3. 若是收定金方违约，交定金方先请求返还定金，然后在违约金和定金罚则中选一个
- 免责事由：不可抗力/情势变更
  1. 不可抗力：迟延履行后发生则不能免责
    + 自然灾害
    + 政府行为
    + 社会异常现象
  2. 情势变更：无法预见、非不可抗力、非属商业风险；可以变更或解除合同

### 第九节 几类主要的有名合同
- 买卖合同：
  1. 动产一物二卖：合同有效，物权归属依据：
    + 一般动产：交付 > 付款 > 合同成立时间
    + 特殊动产（船舶、航空器、机动车）：交付 > 登记 > 合同成立时间（和付款没关系）
  2. 交付的法律效力（无约定或特别规定）：
    + 动产所有权转移
    + 孳息所有权的转移
    + 毁损灭失风险的转移（不论是动产还是不动产）
    + 简易交付，合同生效时交付成就
  3. 标的物的风险负担（非双方原因）：
    + 交付成就之时为分界线
    + 因一方的原因不能按约交付（未依约收取、质量不合格、难以履行后提存等），自违约之日起由违约人承担
    + 出卖人未交付单证资料的不影响标的物风险转移
    + 违约责任和风险负担是独立的
    + 有运输环节：
      1. 已在途后签约的，合同成立之日起风险转移
      2. 出卖人依约代办托运（相当于是以买受人的名义托运）/交付地点无约定or约定不明，交给第一承运人后风险转移
  4. 标的物检验：
    + 不通知视为合格
    + 合理期限/收到之日起2年内（不变期间）
    + 有质量保证期的优先适用质量保证期
  5. 特种买卖合同：
    + 分期付款：
      1. 至少分三次
      2. 到期不支付金额达到全部的20%的，出卖人可请求解除合同or全款都付掉
    + 试用买卖合同：
      1. 试用期满未作表示的，视为同意购买
      2. 不属于试用买卖的：
        - 约定某种检验/检查达标就购买的
        - 约定一定期间内可以调换/退货
    + 以招投标方式订立的买卖合同：招标公告是要约邀请，投标为要约，定标为承诺，书面合同签订时买卖合同正式成立
    + 商品房买卖合同：出卖人必然是 **房地产开发企业**
      1. 广告是要约邀请，但是具体确定的说明允诺是要约，未写进合同也受约束
      2. 预售是特许经营
      3. 被拆迁人有优先权
      4. 有解除权的情形：
        - 质量不合格
        - 面积误差超过 **3%**
        - 一方迟延履行合同义务，催告后 **3个月内** 仍未履行
        - 约定/法定办理期限届满后超过 **1年**，出卖人原因使得无法办理所有权变更登记
      5. 适用惩罚性赔偿金的情形(x5)：出卖人构成欺诈；解除合同赔偿损失后，买受人可要求不超过已付房款 **一倍** 的惩罚性赔偿金：
        - 新设抵押未告知
        - 二卖
        - 没有预售许可
        - 隐瞒已抵押/已售卖/为拆迁补偿安置房屋的情况
      6. 与贷款合同的效力关系
        - 贷款合同未能订立，可以要求解除，有过错方赔偿
        - 买卖合同归于无效，贷款合同也相应解除，双方返还
- 赠与合同：
  1. 单务、无偿、诺成合同，**双方法律行为**
  2. 撤销：任意/法定
    + 任意撤销：
      1. 赠与财产权利转移前可撤销
      2. 经公证/公益性质不得撤销
    + 法定撤销：忘恩行为，返还财产
      1. 赠与人的撤销权：知道后 **1年**
      2. 赠与人的继承人/法定代理人的撤销权：知道后 **6个月**
- 借款合同：货币所有权转移
  1. 计息周期推定1年，提前还的按实际借款期间算利息
  2. 除房贷外，金融机构贷款利率不设上下限
  3. 未按照约定使用贷款，贷款人可以停发、提前收回、解除合同
  4. 利息不能预先扣除；否则按实际借款数还本付息
  5. **民间借贷合同**：new
    + 非金融机构民事主体之间
    + 效力：组织之间/向员工为生产经营需要而订立，有效；无效情形（一方违法犯罪并不当然无效）：
      1. 合同法52条（恶意串通等）
      2. 向金融机构/其他主体/职工集资后转贷，且对方知情
      3. 向非法活动提供贷款且知情
      4. 违背公序良俗及法律法规规定的
    + **利息与利率**：**单利！**
      1. 没有约定：就是没有利息
      2. 约定不明：根据同类确定（自然人之间除外，算没有利息）
      3. 利率范围(年%)：**(24%, 36%]**
        - (0, 24]：支持区
        - (24, 36]：自然债务区，不支持付也不反对还
        - (36, +∞)：无效区，支持返还
      4. 逾期利率：
        - 有约定 in [0%, 24%]：从约定
        - 借期、逾期均未约定：逾期 **6%**
        - 借期约定了，逾期未约定：按借期内利率
        - 可一并主张逾期利率、违约金或其他费用，只要总计不超过年24%
      5. 可以利滚利（前期本金利息一起计入后期本金），只要不超过年24%
    + 自然人之间借款的特别规定：
      1. 为实践合同
      2. 利息无约定或者约定不明确，**均视为无利息**。
    + 法定代表人的责任：
      1. 企业名义借自己用：共同被告/第三人
      2. 个人借企业生产经营用：共同承担责任
- 经营性租赁合同：
  1. 租赁期限超过 **20年** 部分无效
  2. 不定期租赁：双方均可随时解除，合理期限通知
    + 期限6个月以上的，未采用书面形式
    + 未约定/约定不明
    + 期间届满继续使用，出租人无异议
  3. 租金支付周期推定1年
  4. 所有权让与不破租赁
  5. 房屋租赁合同的特别规定：
    + 合同无效情形：违章建筑出租（无规划许可、未批准建设、超过使用期限）
    + 房屋承租人有同等条件优先购买权：
      1. 提前5日通知拍卖
      2. 侵害优先权，可以赔偿损失，不得主张买卖合同无效
      3. 优先权顺位次于共有人和近亲属（直系，配偶父母子女、兄弟姐妹、祖辈四人、孙辈四人）
      4. 通知后 **15日** 未明确表示购买视为放弃
      5. 第三人善意取得后不可对抗
    + 登记备案手续办理与否不影响房屋租赁合同效力
    + 租赁期间承租人死亡，共同居住者可以按原合同租赁房屋
  6. 经同意可以转租，承租人对出租人担责；未经同意转租，出租人可以 **解除合同**
  7. 租赁物危及承租人安全/健康，承租人明知的情况下仍然可以 **随时解除合同**
  8. 毁损灭失风险、维修义务由出租人承担；承租人损害租赁物（不按约定方法/性质使用），出租人可以解除合同并要求赔偿损失
- **融资租赁合同**：书面合同，融资为目的，有点视为承租人所有的意思（只是视为，所有权在出租人）
  1. 占有期间毁损灭失风险、维修、造成他人损害的责任，由承租人承担
  2. 出卖人履行不适当，承租人行使索赔权 **不影响** 其支付租金的义务（除非出租人参与购买决策）
  3. 承租人破产的，租赁物 **不属于** 破产财产
  4. 若承租人无权处分，第三人可善意取得；不适用善意取得的情况（未尽注意义务）：
    + 显著位置做了标识
    + 办理了抵押权登记
    + 第三人没进行融资租赁查询
    + 有证据证明知情
  5. 出租人可要求解除合同的情形：
    + 未经同意无权处分
    + 不付租金达约定情形
    + 无约定的，欠付 **两期以上or总额15%** 以上
  6. 租赁期满所有权归属无法确定的，归出租人
- 承揽合同：加工、修理、定作、测试、检验等
  1. 共同承揽人责任连带
  2. 主要工作外包要同意，辅助工作外包不必；但承揽人总是要负责
  3. 定作人可以随时解除合同，但应当赔偿损失
- **建设工程合同**：特别的承揽合同
  1. 招投标订阴阳合同，以备案的阳合同为结算/争议依据
  2. 无效的情形：
    + 承包人实际没有资质
    + 必须招标而实际并未
  3. 合同无效但竣工了：
    + 验收合格，参照约定付款
    + 验收不合格：
      1. 修复后合格，参照约定付款+承包人承担修复费用
      2. 修复后仍不合格，不付款
  4. 分包：勘察、设计、施工，每块不得肢解后发包
    + 总分承包人责任连带（合同相对性的突破）
    + 主体结构必须由总承包完成
    + 禁止分包人再分包
  5. 承包人垫资：
    + 未约定是垫资，视为工程欠款
      1. 工程欠款利息从约定，无约定按pboc同期同类贷款利率
      2. 工程欠款主要部分是 **合同里写的那些钱！** 其次才是未约定的垫资
    + 约定了垫资：
      1. 未约定利息则没有利息
      2. 约定了利息，高于pboc同期同类贷款利率部分 **无效**
    + 利息从应付工程价款之日起算，推定：
      1. 工程交付之日
      2. 提交竣工结算文件之日
      3. 起诉之日
  6. 实际竣工日期认定：收到竣工结算文件后约定期限内不答复，视作认可
    + 竣工验收合格日
    + 拖延验收的，以提交验收报告之日为竣工日期
    + 未验收擅自使用，以转移占有之日为竣工日期
  7. 发包人不支付建设工程价款：人工费、材料费，不包括违约损失
    + 将工程折价/拍卖，承包人优先受偿
    + 承包人此项优先受偿权 **优于抵押权和其他债权**，但不得对抗已付款的买受人
    + 优先权期限 **6个月**，自竣工日起算
- 委托合同：
  1. 双方均可随时解除，赔偿损失
  2. 可转委托
  3. 隐名代理：委托人A
    + 对方知情，一切照正常代理，此时合同才直接约束委托人和第三人
    + 对方不知情，A/C有不履行的情况，受托人应该向对面披露
    + C不知情时A不履行，B向C披露A后，C可以选择对 **AorB** 主张权利，但是选好了不能改
  4. 应预付处理事务费用和委托报酬（垫资的话要付利息），也可约定无偿等
  5. 损失赔偿：
    + 有偿委托，一般过错赔偿
    + 无偿委托，故意/重大过失才赔偿
  6. 多名受托人共同处理事务的，责任连带
- 技术合同：
  1. 侵害他人技术秘密的技术合同被确认无效后：
    + 善意取得人付钱&保密，可继续使用
    + 恶意则属共同侵权，连带赔偿&保密，不得继续使用
  2. 职务技术成果：
    + 界定：
      1. 履行岗位职责
      2. 离职后 **1年内** 继续从事相关工作，可依约定确认
      3. 主要利用组织的物质技术条件（资金、设备、材料、技术信息等）完成的成果（约定交使用费的、只是用它来测试的除外）
  3. 技术成果的专利申请权归属：
    + 委托开发的：
      1. 申请权属于开发人（交付前不能转让）
      2. 委托人有优先受让权
      3. 开发人取得专利权的，委托人可以免费实施
    + 合作开发的：约定优先，推定：
      1. 开发人共有，有优先受让权
      2. A放弃专利申请权后，其他人申请成功，A可以免费使用
      3. A不同意申请，其他人不能申请
  4. 技术秘密无约定：不区分委托/合作开发，不减少别人的权利即可
    + 各方均可自行使用/授予普通使用许可（非排他性）并独占利益
    + 未经同意转让或授予独占/排他性许可无效

## 第五章 合伙企业法律制度
- 客观题为主，有主观题可能
- 人合制企业，约定最大，例外(x4)：*背诵*
  1. 普通合伙人财产份额出质，必须全体一致同意
  2. 普通合伙人竞业禁止
  3. 普通合伙企业不得约定由部分合伙人获得全部利润/承担全部亏损
  4. 有限合伙人禁止以劳务出资、不能执行合伙事务、不得对外代表有限合伙企业
- 表决方法无约定/约定不明，按法定，无法定则推定一人一票，全体过半数

### 第一节 合伙企业法律制度概述
- 分为普通合伙企业（含特殊的普通合伙企业，律师/会计师事务所等）和有限合伙企业
- 约定最大，下述授权性规范和义务性规范中未注明法律强制的，均为无约定/约定不明时的法定/推定
- 答题时注意写“合伙企业另有约定的除外”等文字

### 第二节 普通合伙企业
- 合伙人无限连带责任
- 设立：
  1. 设立条件：
    + 合伙人数量：两个以上，[2, +∞)
    + 合伙人资格：
      1. 完全民事行为能力的自然人
      2. 法人或其他组织
      3. 国有独资公司、国有企业、上市公司、公益性事业单位和社会团体不得成为普通合伙人（不能被赔光了，但可以成为有限合伙人）
    + 有书面合伙协议，全体合伙人签章后生效（人合制，约定最大；修改办法从约定，推定全体一致同意）
    + 有认缴/实缴的出资，允许的出资形式(4+1)：
      1. 货币
      2. 实物
      3. 知识产权
      4. 土地使用权/其他财产权利
      5. 劳务（仅限普通合伙人）
    + 名称和生产经营场所
  2. 设立登记：营业执照签发日期为合伙企业成立日期（所有企业的成立日期都是这样）
- 财产：认缴的出资&收益&其他；独立性&完整性
  1. 合伙人财产份额转让：
    + 对外：一致同意，其他人有优先购买权
    + 对内：通知即可
  2. 合伙人以财产份额出质：
    + 必须其他合伙人一致同意，否则 **无效**（强制性规定，**不允许约定排除**，不论第三方善恶意）
    + 无效后向善意第三人赔偿损失，不存在善意取得
- 事务执行：
  1. 共同执行or委托若干人执行
    + 执行事务者对外代表合伙企业，其他人不能代表
    + 其他人有监督和查账的权利
    + 可提出异议/撤销委托
    + 争议事项可表决，推定一人一票过半数通过
  2. 应当全体一致同意的事项(6+4)：无约定时的法定 *背诵*
    + 改变名称
    + 改变经营范围和主要经营场所
    + 处分不动产
    + 转让/处分知识产权和其他财产权利
    + 以合伙企业名义为他人提供担保
    + 聘请非合伙人担任经营管理人员
    + 合伙人身份转变
    + 普通合伙人对外转让财产份额
    + 合伙人入伙/决议除名
    + 修改合伙协议
  3. 合伙人义务：
    + 竞业禁止：合伙人不得自营/同他人合作经营与本合伙企业相竞争的业务（强制性规定，**不允许约定排除**）
    + 不得同本企业交易，可约定允许/全体一致同意允许
    + 不得损害本企业利益（恶意串通/侵占财产），退还/赔偿
  4. 决议办法：除若干强制性规定外
    + 按约定
    + 按法定
    + 合伙人一人一票，全体过半数通过
  5. 损益分配原则（利润分配/亏损分担）：
    + 按约定
    + 协商决定
    + 按 **实缴** 出资比例
    + 平均
    + **不得约定** 由部分合伙人获得全部利润/承担全部亏损（**强制性规定**）
  6. 外聘经营管理人员：
    + 可约定or全体一致同意聘请
    + 相当于一种代理，会有无权代理/表见代理/善意相对人问题
- 合伙企业与第三人的关系：
  1. 内部代表权授予约定不得向外对抗（不知情的善意第三人）
  2. 企业的债务清偿：
    + 企业财产
    + 合伙人无限连带
    + 合伙人之间分担和追偿，按损益分配原则来。但内部约定对债权人没有约束力，可找一人/数人/全部/分别，反正就是都可以看心情
  3. 合伙人个人的债务清偿：
    + 与企业无关的债务，对相关债权人的约束：
      1. 不得以其债权抵销对合伙企业的债务
      2. 不得代位行使合伙人在合伙企业中的权利
    + 自有财产不足以清偿：用收益/强制执行财产份额（无先后顺序）
      1. 强制执行份额时，应通知全体合伙人，优先购买权；不得自行接管
      2. 不购买也不同意对外转让，办理退伙结算/结算削减财产份额
- 入伙退伙：
  1. 入伙：
    + 新合伙人入伙，全体一致同意，书面入伙协议，改原合伙协议
    + 对入伙前合伙企业的债务承担 **无限连带责任**
  2. **退伙**：
    + 退伙原因：
      1. 自愿退伙：违规退伙应赔偿
        - 协议退伙：约定的合伙期限内发生以下情形之一
          1. 协议约定的事由出现
          2. 全体一致同意
          3. 发生难以继续参加的事由
          4. 其他合伙人严重违反约定义务
        - 通知退伙：
          1. 未约定合伙期限
          2. 不给企业带来不利影响
          3. 提前 **30日** 通知后可以退伙
      2. 法定退伙：
        - 当然退伙：“被动出问题”，退伙事由实际发生之日生效
          1. 自然人死亡/被宣告死亡（和失踪没关系）
          2. 个人丧失偿债能力
          3. 法人/组织被吊销营业执照、责令关闭、撤销、**被宣告破产**（不是申请或者受理）
          4. 合伙人某种（技术）资格丧失从而不满足法定/约定条件
          5. 全部财产份额被强制执行
          6. 成为无/限制民事能力人，可其他合伙人一致同意转为有限合伙人，不能一致同意则退伙
        - 除名：有下列情形之一，其他人 **一致同意后** 可决议除名；书面除名通知接到生效，如有异议 **30日内** 起诉（但还是生效&退伙了）
          1. 未履行出资义务
          2. 故意重大过失造成企业损失
          3. 执行事务有不正当行为
          4. 发生约定事由
    + 退伙效果：
      1. 财产继承：死亡/被宣告死亡；所有继承人作为一个整体继承
        - 约定/一致同意，继承人取得合伙人资格
        - 退还份额情形：
          1. 继承人不愿意加入
          2. 无法定/约定资格
          3. 其他约定情形
        - 继承人无/限制民事能力人，可经一致同意转有限，不能则退还份额
      2. 退伙结算：
        - 退还办法约定/协商，货币实物均可
        - 按损益分配原则算完后的净值
      3. 退伙人对基于其退伙前原因发生的企业债务无限连带
- 特殊的普通合伙企业：
  1. 企业名称里标明 “特殊普通合伙”
  2. 责任承担：
    + 部分合伙人故意重大过失造成债务的，承担无限/无限连带责任，其他人以 **财产份额** 为限承担责任
    + 故意重大过失合伙人还应赔偿合伙企业损失
    + 其余债务全体无限连带
  3. 防范执业风险：建立执业风险基金、办理职业保险

### 第三节 有限合伙企业
- 普通合伙人和有限合伙人都至少有一个。普通合伙人无限连带；有限合伙人认缴的出资额为限，不执行合伙事务
- 无特殊规定者，适用普通合伙企业的相关规定，下主述特殊规定
- 设立：
  1. 合伙人2个以上，50个以下，[2, 50]
  2. 只剩有限合伙人的应解散，只剩普通合伙人的，应转普通合伙
  3. 有限合伙人 **不得以劳务出资**（强制性规定）
- 事务执行：
  1. 有限合伙人不执行合伙事务，不得对外代表有限合伙企业（**强制性规定**）
  2. 下列行为不视为执行合伙事务，也就是说能做(x8)：*背诵*
    + 参与决定普通合伙人入伙退伙
    + 对经营管理提出建议
    + 参与选择做审计的会计师事务所
    + 获取经审计的财务报告
    + 对涉及自身利益的情况查阅财务资料
    + 在企业中的利益受到侵害时，向有责任的合伙人主张权利/提起诉讼
    + 执行事务合伙人怠于行使权利时督促/为企业利益 **以自己的名义** 提起诉讼（有点股东代表诉讼的意思）
    + 为本企业提供担保
  3. 有限合伙人也有无权代理(无权代表)，第三人善意的问题
    + 第三人有理由相信有限合伙人是普通合伙人并与之交易的，该有限合伙人对该笔交易承担普通合伙人责任，无限连带
    + 无权代理(代表)损失赔偿
  4. 合伙企业利润分配：可以约定将全部利润分配给部分合伙人（此处利润分配可约定优先，未涉及亏损承担；普通合伙一律禁止）
  5. 有限合伙 **人** 权利：不是有限合伙企业的合伙人；当然可以约定排除
    + 可以同本企业进行交易
    + 可以经营与本企业相竞争的业务
- 有限合伙人财产份额出质/转让：
  1. 财产份额可以出质
  2. 可按约定向外转让，**提前30日** 通知其他合伙人，有优先购买权
- 有限合伙人的入伙退伙：
  1. 新入伙的有限合伙人对入伙前债务，以认缴的出资额为限承担责任（前后最多把认缴的全部实缴就行了）
  2. 退伙：
    + 当然退伙情形：比普通合伙人少一条 “个人丧失偿债能力”
      1. 自然人死亡
      2. 法人终止
      3. 丧失相关资格
      4. 全部财产份额被强制执行
    + 丧失民事行为能力：不得因此要求其退伙
    + 死亡后，自然人的继承人/法人的权利承受人取得相应资格
    + 退伙后，对基于其退伙前原因发生的企业债务，以其 **取回的财产承担责任**
- 合伙人身份转变：普通合伙人的无限连带是向前后均追溯的
  1. 普通/有限合伙人身份互变，全体一致同意
  2. 有限变普通，对其有限时的债务 **无限连带**（就像新入伙的普通一样）
  3. 普通变有限，对其普通时的债务无限连带

### 第四节 合伙企业的解散和清算
- 解散情形：
  1. 约定期限届满，决定不再经营
  2. 约定的解散事由出现
  3. 全体合伙人决定解散
  4. 不具备法定人数 **满30日**
  5. 合伙目的已实现/无法实现
  6. 被吊销营业执照/责令关闭/被撤销
- 清算：
  1. 确定清算人：
    + 全体合伙人担任
    + 过半数同意，可在解散事由出现后 **15日内** 指定若干合伙人/第三人
    + 过15日未确定，可申请人民法院指定
  2. 通知/公告债权人：确定清算人之日起算
    + **10日** 内通知债权人；接到后 **30日内** 申报债权
    + **60日** 内在报纸上公告；公告之日起 **45日内** 申报债权
  3. 财产清偿顺序：
    + 清算费用
    + 职工工资、社保和补偿金
    + 税款
    + 普通债务
    + 剩余财产按损益分配原则分配
  4. 钱不够的情况下：
    + 有民事赔偿和罚款罚金并存的，先承担民事赔偿责任
    + 被宣告破产，普通合伙人仍然承担无限连带责任

## 第六章 公司法律制度
- 资合性；with 证券法，大型综合题
- 有限责任公司；股份有限公司（发起设立、募集设立；上市公司）

### 第一节 公司法律制度概述
- 公司对外投资/提供担保：
  1. 按章程规定，由董事会 or 股东(大)会(2选1)作出决议
  2. 公司章程可以规定总额/单项额度限制
  3. 公司为股东/实际控制人提供担保的必须经过股东(大)会决议，回避+出席+表决权过半数(>1/2)
- 法人人格否认制度：滥用法人独立地位/股东有限责任逃避债务，严重损害债权人利益，应负连带责任
  1. 常见理由：
    + 纵向人格混同：公司与股东财产/业务/人员混同
    + 股东违反出资义务/不履行清算义务
    + 横向人格混同：同一控制下的企业之间相互混同
  2. 法律后果：并非彻底否定法人资格，而是在某一具体法律关系中产生效果(x2)
    + 公司行为被视为股东行为，股东承担连带责任
    + 关联公司被视为同一法律主体，相互之间对外部债务承担连带责任
- 出资：允许补正（补充评估、办理变更登记、解除权利负担）
  1. 允许的形式：可以用货币估价&可以依法转让
    + 货币
    + 实物
    + 知识产权
    + 土地使用权
    + 其他财产权利（如股权、债权）
  2. 不允许的形式(x6)：
    + 劳务
    + 信用
    + 自然人姓名
    + 商誉
    + 特许经营权
    + **设定担保的财产**
  3. 以股权出资的要求：权属清楚、权能完整、可依法转让
    + 不得用于出资的情形：(x3+1)
      1. 已设立质权
      2. 股权所在公司章程约定不得转让
      3. 应报批而并未
      4. 其他
    + 须符合的条件：
      1. 出资人合法持有、可转让
      2. 无权利瑕疵、无权利负担
      3. 转让手续已办完
      4. 已进行价值评估
  4. 非货币财产出资的评估作价：
    + 起初未评估，不得直接认定未履行出资义务
    + 人民法院应委托相关机构评估，显著低于章程所载，方可认定未全面履行
    + 事后出资财产贬值，无权要求补足；有约定除外
    + 发起设立的股份有限公司&有限责任公司无需验资；募集设立的股份有限公司的注册资本应当经过验资
  5. 出资财产需要办理权属变更手续的/有权利负担的：
    + 合理期间内办理了权属变更手续/解除权利负担，应认定履行了出资义务
    + 支持实际交付之日起享有股东权利（已交付未登记、已登记未交付）
  6. 以无权处分的财产出资：适用一般无权处分/善意取得制度
    + 公司受让时善意
    + 合理评估作价
    + 已交付/登记
    + 无权处分人赔偿
  7. 以违法犯罪所得出资：以拍卖变卖方式处置，而非直接抽出
  8. 违反出资义务的责任：未履行/未全面履行出资义务
    + 债权人可请求相关股东在未出资本息范围内补充赔偿
    + 设立时股东违反出资义务，**发起人** 与被告股东承担 **连带** 责任（发起人股东的法定资本充实责任），不得约定排除
    + 增资时违反出资义务，**董&高** 未尽忠实勤勉义务者，承担 **相应** 责任
    + 违反出资义务/抽逃出资者，可对其股东权利作合理限制（利润分配、新股优先认购、剩余财产分配）
    + 出资义务（违反/抽逃）**不适用诉讼时效抗辩**
    + 隐名股东/冒名股东：
      1. 名义股东对外承担出资义务，对实际出资人追偿
      2. 冒名行为人对外承担出资义务，被冒名者无责任
  9. 抽逃出资：
    + 合法抽回股本情形(x3)：
      1. 未按期募足股份
      2. 发起人未按期召开创立大会
      3. 创立大会决议不设公司
    + 构成抽逃出资情形(x3+1)：损害公司权益
      1. 虚构债权债务关系转出出资
      2. 虚增利润进行分配
      3. 利用关联交易转出
      4. 其他未经法定程序抽回出资的行为
    + 协助抽逃出资的其他股东、董事、高级管理人员、实际控制人(x4，没有监事)承担 **连带责任**
- 股东资格&权利&义务：
  1. 隐名股东：司法解释只提了有限责任公司的情形
    + 股权代持协议有效，仅是名义股东和实际出资人之间的事；名义股东无权处分/善意取得
    + 实际出资人突破双方合同，要求进入公司内部（转正）：
      1. 参照股东对外转让股权处理
      2. 经公司其他股东半数以上同意（**人数≥1/2**）
  2. 一股二卖：登记对抗/可善意取得
    + 合理信赖公司登记机关的登记
    + 善意/对价/已变更登记，出让人赔偿
    + 未及时变更登记，董&高&实际控制人有过错的，承担相应责任
  3. **股东权利**：资产收益、参与重大决策、选择管理者等
    + 表决权：公司持有的本公司股份没有表决权
    + 知情权：
      1. 有限责任公司：有权查阅&**复制**(x5) + 查阅会计账簿（此项需书面请求，公司15日内书面答复，不服可起诉）
        - 公司章程
        - 股东会会议记录
        - 董事会会议决议
        - 监事会会议决议
        - 财务会计报告
      2. 股份有限公司：有权查阅(x5+2，不能复制)
        - 公司章程
        - 股东大会会议记录
        - 董事会会议决议
        - 监事会会议决议
        - 财务会计报告
        - 股东名册
        - 公司债券存根
    + 新股优先认购权：
      1. 有限责任公司股东默认享有，除非约定放弃/限制
      2. 股份有限公司股东 **默认没有**，除非股东大会决议配售
    + 股利分配请求权：实缴出资比例(有限)/股份比例(股份)，可另外约定比例
    + **异议股东股份回购请求权**：给投反对票股东退出公司的渠道
      1. 有限责任公司情形(x3)：
        - 连续5年盈利&符合利润分配条件，但该5年不分配利润
        - 合并、分立、转让主要财产
        - 章程规定的营业期限届满/其他解散事由出现，决议修改章程使公司存续
      2. 股份有限公司情形：**只有一个**
        - 合并、分立
    + 股东诉讼：代表/直接
      1. **股东代表诉讼**：股东代表公司诉讼，保护公司&股东整体利益
        - 概括：有资格的股东(有限任一；股份连续180日以上单独/合计持有1%以上股份) → 找董/监 → 拒绝/30日/紧急 → 以自己名义直接提起诉讼
        - 具体程序：
          1. 董&高执行职务违规造成损失，有限责任公司的股东（任何一个都有资格）、股份有限公司连续180日以上单独或者合计持有公司1%以上股份的股东，书面请求监事会/无监事会时的监事提起诉讼
          2. 监事违规，书面请求董事会/不设董事会时的执行董事提起诉讼
          3. 外部人士给公司造成损失，书面请求董事会/执行董事/监事会/监事提起诉讼
          4. 收到书面请求后满足下列情形(x3)之一，有资格股东有权为了公司的利益以自己的名义直接提起诉讼
            + 拒绝提起诉讼
            + 自收到之日起 **30日内** 未提起诉讼
            + 情况紧急，不立即提起诉讼将会使公司利益受到难以弥补的损害
      2. 股东直接诉讼：董&高(没有监，监平时不干活)违规损害股东利益(指个别股东的利益)，股东可以提起诉讼
  4. 股东义务：出资、行使股权
    + 违反出资义务：
      1. 股东权利（利润分配、新股优先认购、剩余财产分配权）受限
      2. 未履行出资义务/抽逃全部出资（就是一分钱都没），可以决议解除该股东的股东资格
    + 不得滥用股东权利：合规、不得损害公司/其他股东利益
- 董监高制度：
  1. 高级管理人员：经理、副经理、财务负责人、上市公司董事会秘书以及公司章程规定的其他人员
  2. 不得担任董监高的情形(x5)：违规委任无效，在任时出现应解除职务
    + 无/限制民事行为能力人
    + 因贪污/贿赂/侵占财产/挪用财产/破坏社会主义市场经济秩序(黑五类)，被判处刑罚/剥夺政治权利，执行期满 **未逾5年**
    + 担任破产清算公司/企业的董事/厂长/经理，对破产负有个人责任的，自破产清算完结之日起 **未逾3年**
    + 担任因违法被吊销营业执照/责令关闭的公司/企业的法定代表人(董事长/执行董事/总经理3选1)，并负有个人责任的，自被吊销执照之日起 **未逾3年**
    + 个人所负数额较大债务到期未清偿
  3. 董监高法定义务：忠实、勤勉；包括增资/抽逃出资/一股二卖中的义务
    + 忠实义务：个人/公司利益冲突场合适用，整体股东利益最大化优先
      1. 董&高不得有下列行为(x8)：违规收入归公司所有
        - 挪用公司资金
        - 将公司资金以个人名义存储
        - 违规将公司资金贷出/以公司财产为他人提供担保
        - 违规与本公司订立合同/交易
        - 未经股东(大)会同意，为自己/他人谋取属于公司的商业机会，经营与任职公司同类业务
        - 接受他人与公司交易的佣金为己有
        - 擅自披露公司秘密
        - 其他
      2. 监事不参与经营决策，故忠实义务较轻
      3. 并未一概禁止，一般要求按照公司章程规定、股东(大)会/董事会同意
    + 勤勉义务：尽力干活，适用于无利益冲突的场合
      1. 积极的要求
      2. 消极的抗辩：商业判断规则
- 股东(大)会和董事会决议制度
  1. 决议无法定形式，会议应有会议记录，出席者签名
  2. 决议对决议机构全体成员均有约束力，内部约束不得向外对抗
  3. 决议无效/撤销之诉：内容/程序
    + 股东(大)会/董事会决议内容违反法律/行政法规(人大&国务院)的无效
    + 股东(大)会/董事会召集程序/表决方式违反法律/行政法规/公司章程；决议内容违反公司章程的，股东可以自决议作出之日起 **60日内**，请求法院撤销；股东应提供相应担保

### 第二节 股份有限公司
- 设立：发起设立/募集设立
  1. 注册资本：
    + 发起设立：登记的全体发起人认购的股本总额；何时实缴无规定，发起人缴足前不得新募集股份
    + 募集设立：登记的实收股本总额(必须在设立时全部实缴)，发起人认购的股份不得少于股份总数的 **35%**
  2. 设立程序：
    + 发起设立：首次缴纳出资后，选举董事会/监事会，董事会负责设立登记
    + 募集设立：股款足额缴纳、验资证明出具之日后 **30日内** 召开公司创立大会，提前15日通知，应有过半数股份出席
  3. 设立阶段的合同责任：发起人名义订立，公司追认/实际履行，可由公司承担
- 组织机构：
  1. 股东大会：资合性，股份
    + 职权(x10+1)：
      1. 决定公司的经营方针和投资计划
      2. 选举更换非由职工代表担任的董事、监事，决定有关董事、监事的报酬事项
      3. 审议批准董事会的报告
      4. 审议批准监事会/监事的报告
      5. 审议批准公司的年度财务预算/决算方案
      6. 审议批准公司的利润分配/弥补亏损方案
      7. 对公司增减注册资本作出决议
      8. 对发行公司债券作出决议
      9. 对公司合并、分立、解散、清算、变更公司形式作出决议
      10. 修改公司章程
      11. 公司章程规定的其他职权
    + 上市公司股东大会额外职权：
      1. 聘用/解聘会计师事务所
      2. 审议公司一年内购买/出售重大资产(超过公司最近一期经审计总资产30%)的事项
      3. 审议批准变更募集资金用途的事项
      4. 审议股权激励计划
      5. 审议批准下列对外担保行为(x5)：
        - 本公司&控股子公司对外担保总额达到/超过最近一期经审计 净资产50% 以后提供的任何担保
        - 对外担保总额达到/超过最近一期经审计 总资产30% 以后提供的任何担保
        - 为借款后资产负债率超过70%的对象提供的担保
        - 单笔担保额超过最近一期经审计 净资产10% 的担保
        - 对股东/实际控制人及其关联方提供的担保
    + 会议形式：年会/临时大会
      1. 年会：每年召开一次；上市公司应于上一会计年度结束后 **6个月内** 举行
      2. 临时大会召开情形(x5+1)：满足其一后，应在两个月内召开
        - 董事人数不足法定最低人数 or 章程规定人数的 **<2/3** 时
        - 未弥补的亏损达实收股本总额 **≥1/3** 时
        - 单独或者合计持有 **10%以上** 股份的股东请求时
        - 董事会认为必要时
        - 监事会提议召开时
        - 章程规定的其他情形
      3. 会议的召集：
        - 董事会召集，主持人：董事长 → 副董事长 → 半数以上董事共同推举一名董事主持
        - 董事会撂挑子 → 监事会召集&主持 → 连续90日以上单独或者合计持有公司10%以上股份的股东自行召集&主持
      4. 会议的通知：时间/地点/审议事项
        - 年会：召开前20日通知
        - 临时大会：召开前15日通知
        - 发行无记名股票：召开前30日公告
        - 不得对通知中未列明的事项作出决议
      5. 提案：
        - 单独或者合计持有 **3%以上** 股份的股东，可在召开10日前提出临时提案书面提交董事会
        - 董事会收到提案后2日内通知其他股东
      6. 表决和决议事项：资合性；出席会议的
        - 每一股份有一表决权，公司持有的本公司股份没有表决权
        - 法律 **未规定** 出席股东大会最低人数和持股比例要求
        - 普通决议：出席会议的股东所持表决权 **过半数(>1/2)** 通过
        - 特别决议(x4)：出席会议的股东所持表决权 **2/3以上** 通过
          1. 修改公司章程
          2. 增减注册资本
          3. 合并、分立、解散
          4. 变更公司形式
      7. 累积投票制：控股股东比例30%以上的上市公司应采用累积投票制
      8. 会议记录：主持人、出席会议的董事签名（股东人太多就不签了）
  2. 董事会：
    + 人数：5至19人，**[5, 19]**；可以有职工代表（≥0，职工大会/职工代表大会选举产生）
    + 董事任期：由章程具体规定，每届不得超过 **≤3年**，可以连任
    + 董事会职权(x10+1)：制订方案/决定，对股东大会负责
      1. 召集股东大会会议，向股东大会报告工作
      2. 执行股东大会的决议
      3. 决定公司的经营计划和投资方案
      4. 制订公司年度财务预算方案、决算方案
      5. 制定公司利润分配/亏损弥补方案
      6. 制定公司增减注册资本、发行公司债券的方案
      7. 制订公司合并、分立、解散、变更公司形式的方案
      8. 决定内部管理机构的设置
      9. 决定聘任/解聘公司经理及其报酬，并根据经理的提名聘任/解聘副经理/财务负责人及其报酬
      10. 制定公司基本管理制度
      11. 章程规定的其他职权
    + 机构设置：
      1. 董事长一人，可以设副董事长：全体董事过半数选举产生
      2. 履行职务：董事长 → 副董事长 → 半数以上董事共同推举一名董事
      3. 上市公司董事会可下设专门委员会（战略、审计、提名、薪酬考核等）：
        - 全部由董事组成
        - 审计/提名/薪酬考核委员会中独立董事应占多数(≥1/2)并任召集人
        - 审计委员会至少一名独立董事是会计专业人士
    + 会议召开：
      1. 每年至少 **2次**，召开 **10日前** 通知全体董&监
      2. 临时会议提议(x3)：董事长接到后10日内召集&主持，也可另约定
        - 代表10%以上表决权的股东
        - 1/3以上董事
        - 监事会
      3. 过半数董事出席方可举行，可书面委托其他董事代为出席（委托他人也算出席）
      4. 一人一票，全体董事过半数通过（不是出席的）
    + 会议记录：出席会议董事签名，监事不用签名；决议违规损害公司，赞成者赔偿/异议者免责
  3. 经营管理机关：经理
    + 上市公司总经理必须专职，不得在股东单位担任董事以外的职务
    + 股份有限公司不得直接/通过子公司向董监高提供借款
    + 上市公司高级管理人员(3+1)必须在上市公司领薪，不得由控股股东代发薪水
  4. 监事会：
    + 组成和机构设置：
      1. 不得少于 **≥3人**，应当包括股东代表和职工代表
      2. 职工代表比例不得低于 **≥1/3**，职工大会/职工代表大会选举
      3. 董&高不得兼任监事
      4. 主席一人，可以设副主席；全体监事过半数选举产生
      5. 履行职务：主席 → 副主席 → 半数以上监事共同推举一名监事
    + 职权(x6+1+2)：列席董事会会议提质询/建议；调查经营异常
      1. 检查公司财务
      2. 对董&高工作进行监督，对违反法律/行政法规/公司章程/股东会决议者提出罢免建议
      3. 董&高损害公司利益时要求纠正
      4. 提议召开临时股东大会，在董事会不召集主持时顶上
      5. 向股东大会提出提案
      6. 参与股东代表诉讼
      7. 章程规定的其他职权
    + 任期：每届 **=3年**，可以连任
    + 会议召开：每 **6个月** 至少召开一次会议
  5. **上市公司组织机构的特别规定**：
    + 增加股东大会特别决议事项(x4+3)：with 证券法，出席 + ≥2/3
      1. 上市公司在1年内购买/出售重大资产/担保金额超过公司资产总额 **>30%** 的
      2. 重大资产重组
      3. 发行证券(x4, x10+N)
    + 设立独立董事
    + 设董事会秘书，属于高级管理人员
    + 增设关联关系股东/董事表决权排除制度：
      1. 关联关系股东回避+出席+≥2/3，见证券法
      2. 关联关系董事不得表决/代理表决，过半数无关联关系董事出席即可举行；决议经全体无关联关系董事过半数通过；出席的无关联关系董事不足 **<3人**，提交股东大会审议
    + 允许股权激励机制
- **上市公司独立董事制度**：独立履职，保护中小股东
  1. 董事会成员中至少 **≥1/3** 为独立董事
  2. 任职条件：
    + 具备担任上市公司董事资格（不得担任的5种情形：欠缺行为能力/黑五类5年/破产3年/公司违法3年/欠较多钱不还）
    + 具有5年以上法律/经济/其他履职必须的工作经验
    + 有独立性/熟悉相关知识&法律法规/章程规定的其他条件
    + **不得担任独立董事** 的情形(x5+1)：(任职/自然人股东1%or前10/法人股东5%or前5)x(目前+最近1年)+专业服务人员
      1. 在上市公司或其附属企业任职(不论任何职位)的人员及其直系亲属&主要社会关系(直系亲属：配偶父母子女；主要社会关系：兄弟姐妹及其配偶、配偶的父母&兄弟姐妹、子女的配偶)
      2. 直接/间接持有上市公司已发行股份1%以上or上市公司前10名股东中的自然人股东及其直系亲属
      3. 在直接/间接持有上市公司已发行股份5%以上or上市公司前5名股东单位任职(不论任何职位)的人员及其直系亲属
      4. 最近1年内曾经具有前三项情形者
      5. 为上市公司或其附属企业提供财务/法律/咨询等服务的人员
      6. 公司章程/证监会认定的其他人员
  3. 候选人提名资格：给股东大会选举
    + 董事会
    + 监事会
    + 单独或者合计持有上市公司已发行股份1%以上的股东
  4. 任期：与其他董事相同(≤3y)，可以连任，总长不得超过6年
  5. 特别职权(x6)：一般董事职权之外，行使特别职权应取得 **全体独立董事1/2以上同意**
    + 认可重大关联交易(总额>300w or >最近经审计净资产5%)，然后提交董事会讨论
    + 提议聘用/解聘会计师事务所
    + 向董事会提请召开临时股东大会
    + 提议召开董事会
    + 独立聘请外部审计/咨询机构
    + 股东大会前公开征集投票权
  6. 对重大事项发表独立意见：同意/保留理由/反对理由/无法发表意见及障碍
    + 提名任免董事/聘任解聘高级管理人员/董&高薪酬
    + 股东/实际控制人及其关联企业对现有/将有的重大借款(总额>300w or >最近经审计净资产5%)及回收措施
    + 认为可能损害中小股东权益的事项
    + 章程规定的其他事项
  7. 撤换和辞职：
    + 连续3次未亲自出席董事会会议，董事会应提请撤换
    + 不得无故免职/特别披露事项/可公开声明免职理由异议
    + 可书面辞职/对相关情况说明/低于法定人数则补缺后生效
- 股份发行和转让：
  1. 优先股：
    + 已发行优先股不得超过普通股股份总数的≤50%；筹资金额不得超过发行前净资产≤50%
    + 优先股股东权利：
      1. 按约定的股息率优先分配利润，完全支付前不得向普通股股东分配；必须现金股息
      2. 优先分配剩余财产
    + 参与决策权利受限，仅以下情形有表决权(x4+1)：其余时间不出席会议&无表决权
      1. 修改公司章程中与优先股有关的内容
      2. 单次/累计减少注册资本超过10%
      3. 公司合并/分立/解散/变更公司形式
      4. 发行优先股
      5. 章程规定的其他情形
    + 优先股股东参与表决事项的决议应分2组，分别 **出席+≥2/3** 通过：
      1. 普通股股东 + 表决权恢复的优先股股东
      2. 优先股股东（不含表决权恢复的）
    + 优先股表决权恢复：累计3个会计年度/连续2个会计年度未按约定支付优先股股息
      1. 股息可累积的优先股，全额支付所欠股息为止
      2. 股息不可累积的优先股，全额支付当年股息为止
      3. 每股优先股享有的表决权依公司章程规定，可规定表决权恢复的其他情形
    + 计算持股比例时，仅计算普通股和表决权恢复的优先股的事项(x4)：
      1. ≥10%股东请求召开临时股东大会
      2. 连续90日≥10%股东自行召集主持股东大会
      3. ≥3%股东提交股东大会临时提案
      4. 控股股东的认定：≥50% or 足以产生重大影响
  2. 股份转让的限制(x6)：原则上自由转让；基于法律规定/公法行为等除外(继承/遗赠/强制执行/依法分割财产等)
    + 转让场所
    + 发起人持有的本公司股份自公司成立之日起1年内不得转让
    + 公开发行前已发行的股份，自上市之日起1年内不得转让
    + 上市公司董监高转让/买卖本公司股份限制：
      1. 转让股份限制：限制出售
        - 任职期间每年转让不得超过所持 **≤25%**，所持不超过1,000股的不受该比例限制
        - 自公司上市之日起1年内不得转让
        - 离职后半年内不得转让
        - 公司章程规定的更多限制
      2. 下列期间不得买卖本公司股份：买进卖出都不行
        - 定期报告公告前30日内
        - 业绩预告、业绩快报公告前10日内
        - 自重大事项发生之日/决策过程中，至披露后2个交易日内
        - 证交所规定的其他期间
    + 公司一般不得收购自身股份，下列情形除外(x4)：收购之日起限期处分；1-3项要求股东大会决议
      1. 减少注册资本；10日内注销
      2. 与持有本公司股份的其他公司合并；6个月内转让或注销
      3. 将股份奖励给职工；不得超过已发行股份总额≤5%，用税后利润支付，1年内转让给职工
      4. 异议股东请求回购股份（股份有限公司仅合并/分立）；6个月内转让或注销
    + 不得接受本公司股票作为质押权的标的

### 第三节 有限责任公司
- 设立：
  1. 人数：50个以下，允许一人公司，[1, 50]
  2. 注册资本为登记的全体股东认缴的出资额，没有最低资本限额和出资期限
- 组织机构：
  1. 股东会：人合+资合，表决权
    + 职权(x10+1)：不包括同意对外转让股权；完全同股份有限公司；可书面形式一致同意代替召开股东会
    + 会议形式：定期会议/临时会议
      1. 定期会议：按公司章程规定，一般一年一次
      2. 临时会议召开提议人：2个月内召开
        - 10%以上表决权的股东
        - 1/3以上的董事
        - 监事会/不设监事会的监事
    + 会议召集：
      1. 首次会议由出资最多的股东召集和主持（法定事项）
      2. 董事会/执行董事召集，主持人：董事长 → 副董事长 → 半数以上董事推举一名董事主持
      3. 召集&主持责任传递：董事会/执行董事 → 监事会/监事 → 10%以上表决权的股东
    + 通知/记录：15日以前通知，可另约定；会议记录全体股东签名
    + 决议：全部表决权为分母，普通决议无法定比例
      1. 表决权：按出资比例行使表决权，**可在公司章程中另行规定**
      2. 特别决议(x4, 均同股份公司)：≥2/3以上表决权通过（全部表决权，不是出席会议的）
        - 修改公司章程
        - 增减注册资本
        - 合并/分立/解散
        - 变更公司形式
      3. 普通决议：由公司章程规定，**法律无任何规定**
  2. 董事会：
    + 组成：
      1. 人数：3至13人，[3, 13]；股东较少/规模较小的可不设董事会，设一名执行董事
      2. 职工代表：董事会看“国有”，监事会是“所有”
        - 大部分都随意(≥0)
        - 国有独资公司、两个以上国有企业/国有投资主体设立的有限责任公司，董事会中应当有职工代表(≥1, 无数量规定)；职工(代表)大会选举产生
      3. 董事长一人，可以设副董事长；产生办法由章程规定
    + 任期/职权/召集：与股份有限公司相同(≤3y可连任, 10+1, 3层甩锅)
    + 表决程序：一人一票，其他由章程规定（注意没有法定全体过半数）
  3. 经理：是选设机构；如设经理，职权与股份有限公司相同
  4. 监事会：
    + 组成：
      1. 人数：设监事会的，不得少于≥3人；股东较少/规模较小可以不设监事会，设1-2名监事
      2. 职工代表：设监事会的，职工代表比例不得低于≥1/3；不设监事会的不用在意
      3. 监事会设主席一人，全体过半数选举产生
      4. 董&高不得兼任监事
    + 任期/职权：与股份有限公司相同(=3y可连任, 6+1+2)
    + 召集/决议：
      1. 履职甩锅2层（监事会主席 → 半数以上推举）
      2. 每年度至少召开1次会议；监事可以提议召开临时会议
      3. 决议应当经半数以上监事通过
- 一人有限责任公司的特别规定：股东是一个自然人/法人
  1. 股东：对一个自然人严格约束，一孩政策&断子绝孙
    + 一个自然人只能投资设立1个一人有限责任公司，且该一人有限责任公司不能投资设立一人有限责任公司
    + 对源头是法人的一人有限责任公司没有此限制
    + 工商登记/营业执照中应注明自然人/法人独资
  2. 组织机构：
    + 不设股东会，原属股东会决策必须书面作出，签字后置备于公司
    + 公司章程必须制定
  3. 审计：每一会计年度终了时必须编制财务会计报告，并经审计
  4. 法人人格否认原则：股东不能证明公司财产独立于自己财产的，应对公司债务承担连带责任（证明责任在股东）
- 国有独资公司的特别规定：国家单独出资、国有资产监督管理机构(下简称国资委)履行出资人职责的有限责任公司
  1. 公司章程由国资委制定or董事会制定后报批
  2. 组织机构：
    + 不设股东会：
      1. 国资委行使相应职权，可部分授权董事会
      2. 必须由国资委决定的事项：
        - 合并/分立/解散
        - 增减注册资本
        - 发行公司债券
        - 利润分配
    + 董事会：
      1. 应当有职工代表(≥1)；其余由国资委委派
      2. 董事长一人，可以设副董事长；均由国资委指定
    + 设经理，可由董事兼任
    + 董&高未经国资委同意，不得在其他公司/企业/经济组织兼职
    + 监事会：
      1. 不得少于≥5人；职工代表比例不得低于≥1/3
      2. 非职工代表由国资委委派，主席由国资委指定
- **有限责任公司的股权转让**：约定最优先；法定体现一定人合性
  1. 股东之间转让，协商一致即可
  2. 对外转让：无须股东会决议
    + 应当经其他股东过半人数(>1/2)同意(唯一看股东人数的)：未过半数同意，异议股东应当购买该股权，不购买的视为同意
    + 应书面通知其他股东征求同意：接到之日起满30日未答复，视为同意
    + 经同意转让的，其他股东有优先购买权；两人以上同时主张优先，协商 → 按转让时各自(对公司的)出资比例行使
  3. 法院强制执行：通知公司&全体股东，有优先购买权；通知之日起20日内不行使视为放弃
  4. 自然人股东死亡后，继承人可直接取得股东资格
  5. 转让程序：
    + 注销原股东出资证明书
    + 向新股东签发出资证明书
    + 修改公司章程/股东名册的记载（此项修改不需要股东会表决）
    + 办理工商变更登记
  6. 异议股东股份回购请求权：3种情形
    + 可自股东会决议通过之日起60日内与公司达成股权收购协议
    + 否则可自决议通过之日起90日内提起诉讼要求回购
- 组织形态变更：
  1. 常见有限变股份，证监会认可IPO持续经营条件从有限成立之日起算
  2. 债权债务承继
  3. 有限变股份，要求折合的实收股本总额不得高于(≤)原公司 **净** 资产额

### 第四节 公司的财务会计
- 股份有限公司财务会计报告应于召开股东大会年会的20日前置备于公司
- 利润分配顺序：
  1. 在税法规定的期限内(5年)，弥补以前年度亏损
  2. 缴纳所得税
  3. 弥补用税前利润弥补后仍存在的亏损
  4. 提取法定公积金
  5. 提取任意公积金
  6. 向股东分配利润（按出资/股份比例，可另约定）
- 公积金：盈余公积/资本公积；10%/25%/50%
  1. 盈余公积金：从税后利润(A)中提取的，法定/任意
    + 法定公积金：税后利润的10%；累计额为注册资本50%以上时可不再提取(10%·A)
    + 任意公积金：按股东(大)会决议比例提取，被乘数和法定公积金一样(n%·A)
  2. 资本公积金：直接由资本原因形成，如股票发行溢价款等
  3. 主要用途(x3)：*背诵*
    + 弥补公司亏损（税前利润 → 税后利润 → 盈余公积金；资本公积金不得用于弥补亏损）
    + 扩大经营规模
    + 转增公司资本（用任意公积金没有限制；用法定公积金时，转增后留存的不得少于(≥)转增前注册资本的25%）

### 第五节 公司合并、分立与减资
- 10-30, 30-45
- 合并：法定合并，按法定程序进行，不需要经过清算程序；吸收合并(A+B→A) or 新设合并(A+B→C)
  1. 法定合并的便利(x3)：可能损害债权人/股东利益，故须严格遵守法定程序
    + 被消灭公司的债务转移不需要债权人的同意，直接由合并后的公司承继
    + 被消灭公司不需要经过清算程序
    + 股东(大)会通过即可，不需要征求每个股东意见(因合并是公司行为)
  2. 法定程序(x5)：
    + 签订合并协议
    + 编制资产负债表和财产清单
    + 参与公司 **各自** 作出合并决议（股东(大)会特别决议）
    + 通知债权人：决议作出之日起通知/公告，债权人可以要求公司清偿债务or提供相应担保
      1. 10日内通知债权人，接到后30日内
      2. 30日内公告，公告之日起45日内
    + 办理工商变更/注销/设立登记（上一步完成后，即公告之日起45日后）
  3. 异议股东有回购请求权（有限60日/90日；股份6个月内转让/注销）
- 分立：法定分立/法定程序/无须清算；派生分立(A→A+B) or 新设分立(A→B+C)；与合并基本同理
  1. 便利(3+1)：
    + 公司消灭无须清算
    + 股权变化无须全体同意
    + 无需经过法定减资程序（分立程序已保护了债权人）
    + 债务转移未经债权人同意/认可不对其发生效力，即默认分立后的公司对原债务 **承担连带责任**
  2. 程序(x5)：
    + 签分立协议
    + 编表/清单
    + 各自作出分立决议(≥2/3)
    + 通知债权人：10日内通知/30日内公告；债权人 **没有** 请求清偿债务/提供担保的权利
  3. 债权人保护：除非分立前与债权人达成书面协议约定，否则分立后的公司责任连带；不得用内部约定对抗
  4. 异议股东有回购请求权
- 减资：
  1. 程序(x4)：
    + 股东(大)会作出减资决议/修改章程（特别决议，≥2/3）
    + 编表/清单
    + 通知债权人：10-30, 30-45；有权请求清偿债务/提供担保（同合并）
    + 办理减资登记（公告45日后）
  2. 异议股东 **没有** 回购请求权

### 第六节 公司解散和清算
- 解散：
  1. 除因法定合并/分立而解散不必清算，其他解散必须经过法定清算程序
  2. 强制解散：打破僵局的措施
    + 经营发生严重困难/存续会造成重大损失/其他途径不能解决，全部表决权10%以上的股东可以提起解散公司诉讼
    + 条件(x4)：存在以下事由之一，经营管理发生严重困难，法院应予受理解散诉讼
      1. 持续2年以上无法召开股东(大)会
      2. 持续2年以上无法作出有效股东(大)会决议
      3. 董事长期冲突，无法由股东(大)会解决
      4. 其他严重困难，公司存续会使股东利益受到重大损失的情形
    + 提起解散公司诉讼相关规定：
      1. 不得以股东权利(知情权/利润分配请求权等)受到损害、公司亏损/资不抵债、被吊销执照未清算等为由提起解散公司诉讼
      2. ≥10%股东提起解散公司诉讼应以公司为被告，其他股东为第三人；应告知其他股东
      3. 应注重调解；经调解公司收购原告股份的，应自调解书生效之日起6个月内将股份转让/注销
- 清算：解散or被宣告破产后的程序；结束公司事务、清算完毕注销后消灭主体资格
  1. 清算组的成立：
    + 解散事由出现之日起15日内成立
    + 有下列情形(x3)，适格主体(分号后注明)可申请法院指定清算组：
      1. 逾期(>15日)不成立清算组；债权人
      2. 清算组成立但故意拖延清算的；债权人&公司股东
      3. 违法清算，可能严重损害债权人/股东利益；债权人
  2. 清算期间公司行为限制：代表机构为清算组，仍以公司名义进行诉讼
  3. 清算组组成：
    + 有限责任公司：由股东组成
    + 股份有限公司：由董事或者股东大会确定的人员组成
  4. 清算程序：10-30, 60-45
    + 通知债权人：清算组成立之日起10日内书面通知债权人；60日内在全国/省级有影响的报纸上公告
    + 债权申报和登记：
      1. 接到通知30日内；公告之日起45日内申报债权
      2. 逾期补充申报的可在尚未分配财产中受偿；不接受因重大过错逾期的
    + 清理财产，制定清算方案
      1. 发现资不抵债的应申请宣告破产
      2. 自行清算的，清算方案报股东(大)会决议确认；法院组织清算的，方案报法院确认
    + 清偿债务
      1. 清算费用
      2. 职工工资/社保/补偿金
      3. 税款
      4. 普通债务
      5. 分配给股东
    + 注销登记，公告公司终止
  5. 人民法院组织清算的，应自清算组成立之日起6个月内清算完毕

## 第七章 证券法律制度
- with 公司法，大型综合题
- 2015 new 公司债券、上市公司收购、上市公司重大资产重组
- 2016 new IPO、上市公司收购、非上市公众公司

### 第一节 证券法律制度概述
- 构成公开发行的情形(x2+1)：
  1. 向不特定对象发行证券的（对购买人没有资质要求/随机确定）
  2. 向特定对象发行证券累计超过200人的（特定对象主要指合格投资者）
  3. 采用广告、公开劝诱或变相公开方式宣传发行的
- 持续信息披露：
  1. 定期报告：发行股票/债券的公司；年度/中期/季度报告；公比1/2
    + 年度报告：会计年度结束之日起4个月内披露
    + 中期报告：会计年度上半年结束之日起2个月内披露
    + 季度报告：会计年度第3/9个月结束后1个月内披露
  2. 临时报告：**重大事件**(x20+1, 7/8/14)
    + 经营方针/经营范围重大变化
    + 重大投资行为/购置资产决定
    + 订立重要合同，可能对资产/负债/权益/经营成果有重要影响
    + 重大债务/重大违约/大额赔偿责任
    + 重大亏损/重大损失
    + 生产经营外部条件重大变化
    + (\*)公司(任何一个)董事、1/3以上监事、经理发生变动；董事长/经理无法履行职责
    + (\*)持有公司5%以上股份的股东/实际控制人，持股/控股情况发生较大变化
    + 减资/合并/分立/解散/申请破产的决定，或进入破产程序/被责令关闭
    + 重大诉讼/仲裁；股东大会/董事会决议被撤销/宣告无效
    + 公司/董监高涉嫌违法违规被调查/处罚/采取强制措施
    + 新公布的政府/行业法规政策会对公司产生重大影响
    + 董事会就发行新股/再融资/股权激励方案形成决议
    + (\*)法院裁决禁止控股股东转让所持股份；任一股东所持5%以上股份被质押/冻结/司法拍卖/托管/设定信托/限制表决权
    + 主要资产被查封/扣押/冻结/抵押/质押
    + 主要/全部业务陷入停顿
    + 对外提供重大担保
    + 获得大额营业外收入
    + 变更会计政策/会计估计
    + 更正前期披露错误
    + 证监会规定的其他情形
  3. 披露时点：重大事件最先发生(3选1)/已经出现泄露情形(x3)，发生日起 **2个交易日内**
    + 最先发生的任一时点(3选1)：
      1. 董/监事会形成决议时
      2. 签署意向书/协议时
      3. 董监高知悉该事件发生并报告时
    + 上述时点之前出现下列情形也应及时披露(x3)：
      1. 难以保密
      2. 已经泄露或者市场出现传闻
      3. 证券及衍生品出现异常交易
- 董监高/股东/实际控制人在信息披露中的职责：
  1. 董监高：
    + 董&高对定期报告签署书面确认意见
    + 监事会审核定期报告并出具书面审核意见（注意是监事会）
  2. 股东/实际控制人，发生以下事件(x3+1)应主动告知董事会并配合披露：
    + (重大事件8#)持有5%以上股份的，持股/控股情况较大变化
    + (重大事件14#)法院裁决禁止转让所持股份；所持5%以上股份被质押/冻结/司法拍卖/托管/设定信托/限制表决权
    + 拟对上市公司进行重大资产/业务重组
    + 证监会规定的其他情形

### 第二节 股票的发行
- 主要类型(3大类)：
  1. 非公众公司非公开发行/转让，股东未超(≤)200人：由公司法规制，不属于证券法管辖
  2. 非公众公司公开发行/转让；非上市公司定向发行：非上市公众公司
  3. 首次公开发行(IPO)股票并上市/上市公司增发股票：上市公司常规
- **非上市公众公司**：2016 new >200/公开转让；不能向不特定对象发行；定向发行和公开转让的资格是分别核准的
  1. 概念：具有特定情形之一(x2)的未上市股份有限公司；仅有证券法意义；全国中小企业股份转让系统(“新三板”)作为其股票发行&交易的市场
    + 股票向特定对象发行/转让导致股东累计超过(>)200人的
    + 股票公开转让（不论股东人数）
  2. 普通公司成为非上市公众公司的三种途径：
    + (A)定向发行后超200人
    + (B)定向转让后超200人
    + (C)申请公开转让
  3. 不同情形下的核准：3种≤200人豁免核准/申请(普通公司非公开转让人数超过又降回/股东200人以下普通公司申请公开转让/公众公司定向发行后股东200人以下)
    + 起初是普通公司：股份有限公司发起人[2, 200]，募集设立实务中没有；所以新设普通公司没有超过200人的，历史遗留超过200人的参照解决
      1. 通过途径A：定向发行1#
        - 应经过证监会核准
        - 发行人董事会决议后提请股东大会特别决议批准
      2. 通过途径B：
        - 发生之日起3个月内申请核准
        - 3个月内将股东人数降至200人以下的可以不提出申请（豁免核准/申请1#），还是普通公司
      3. 通过途径C：
        - 董事会决议后提请股东大会特别决议批准
        - 若股东人数未超过200人则豁免核准（豁免核准2#），新三板自律管理；其余需要证监会核准
    + 已经是非上市公众公司：此时向特定对象发行，为定向发行2#；并不直接取得公开转让资格
      1. 股东超过200人
        - 向特定对象发行股票：需要证监会核准
        - 申请公开转让：需要证监会核准；董事会决议后提请股东大会特别决议批准
      2. 股东没超过200人（途径C来的，已公开转让；故只需讨论定向发行即可）
        - 定向发行后股东超过200人：需要证监会核准
        - 定向发行后股东不超过200人：豁免核准3#，新三板自律管理
  4. 监管要求：强制信息披露义务，经审计定期报告
    + 股票公开转让&定向发行的：定期披露半年度报告和年度报告
    + 向特定对象转让股东超过200人的：仅需定期披露年度报告
  5. 向特定对象发行股票（定向发行）：
    + 定向发行的特定对象范围：(2)+(3)合计不得超过 **≤35** 名；非公开转让的特定对象没提
      1. 公司股东
      2. 公司董监高&核心员工（董事会提名→员工&监事会意见→股东大会批准）
      3. 符合投资者适当性规定的自然人/法人/其他经济组织
    + 包含两种情形：普通公司定向发行后股东超过200人；非上市公众公司向特定对象发行
    + 发行人董事会应就发行方案作决议，并提请股东大会特别决议通过
    + 可以一次核准，分期发行：
      1. 核准之日起3个月内首期发行，不少于总发行数≥50%
      2. 核准之日起12个月内将剩余数量发完，各期数量自行确定
      3. 每期发行后5个工作日内报证监会备案
      4. 超有效期未发行作废，须重新核准
  6. 新三板投资者适当性标准：
    + 满足条件(x3)之一的机构投资者：
      1. 注册资本500万元以上的法人
      2. 实缴出资500万元以上的合伙企业
      3. 信托/基金/银行理财产品/资产管理计划等
    + 同时符合下列条件的自然人投资者：
      1. 本人名下前一交易日终证券类资产市值500万元以上
      2. 具有2年以上证券投资经验(本人名下账户发生首笔股票交易之日起算) or 具有会计/金融/投资/财经专业背景
- **首次公开发行股票并上市**：
  1. 条件：股份有限公司；主板&中小板/创业板
    + 主板&中小板IPO条件
      1. 存续期限：3年以上
        - 股份有限公司自成立后持续经营3年以上
        - 有限责任公司按账面净资产折股整体变更为股份有限公司，可从有限成立之日起算3年
        - 国务院可另批准
      2. 注册资本足额缴纳，无重大权属纠纷
      3. 经营合法
      4. 最近3年主营业务、董&高没有重大变化；实际控制人未变更
        - 表决权比例最高者发生变化，且变化前后股东不属于同一实际控制人，视为控制权发生变更
        - 控制权归属难以判断/无人具有控制权时，具有以下情形(x3)可视为控制权未变更：
          1. 股权/控制结构/管理层/主营业务最近3年无重大变化
          2. 股权/控制结构不影响公司治理有效性
          3. 发行人/保荐人/律师能提供充分证据证明
      5. 股权清晰，控股股份无重大权属纠纷
      6. 治理结构良好
      7. 具有持续盈利能力，不得有下列情形(x5+1)：
        - 经营模式/产出结构已经/即将重大变化，对持续盈利能力构成重大不利影响
        - 行业地位/行业经营环境已经/即将重大变化，对持续盈利能力构成重大不利影响
        - 最近1个会计年度营业收入/净利润对关联方/存在重大不确定性的客户有重大依赖
        - 最近1个会计年度的净利润主要来自合并财报以外的投资收益
        - 在用的商标/专利/专有技术/特许经营权等重要资产/技术的取得/使用存在重大不利变化风险
        - 其他
      8. 财务状况良好：
        - 财务管理规范，无保留意见审计报告
          1. 审计报告意见类型共4种：无保留意见；保留意见；否定意见；无法表示意见
          2. 可附带文字段落有2种：带强调事项段；带其他事项段
        - 财务指标良好(x5)：
          1. 最近3个会计年度净利润均为正且累计超过>3,000万元（净利润取扣除非经常性损益前后较低者）
          2. 最近3个会计年度经营活动产生的现金流量净额累计超过5,000万元 or 营业收入累计超过3亿元
          3. **发行前** 股本总额不少于≥3,000万元
          4. 最近一期期末无形资产占净资产比例不高于≤20%（扣除土地使用权/水面养殖权/采矿权等）
          5. 最近一期期末不存在未弥补亏损
        - 依法纳税，收入对税收优惠无严重依赖
        - 不存在重大偿债风险和重大担保/诉讼/仲裁等或有事项
        - 财务资料真实完整；招股说明书中引用的财务报表在其最近一期截止日后6个月内有效，特别情况可申请延长不超过1个月
      9. 不存在法定障碍(x5+1)：
        - 最近3年内未经核准公开/变相公开发行证券or虽发生在36个月前但目前仍持续
        - (\*)最近3年内违反工商/税收/土地/环保/海关或其他法律/行政法规，情节严重受到行政处罚
        - 最近3年内曾向证监会提出发行申请，但报送文件虚假陈述/骗取核准/干扰审核/仿造签章
        - 本次报送的申请文件虚假陈述
        - (\*)涉嫌犯罪被立案侦查，尚未有明确结论意见
        - 其他
    + 创业板IPO条件：稍低
      1. 股份有限公司持续经营3年以上；有限变股份可从有限成立之日起算
      2. (\*)最近2年连续盈利且净利润累计不少于1,000万元 or 最近1年盈利且营业收入不少于5,000万元；净利润取扣除非经常性损益前后孰低者
      3. 最近一期期末净资产不少于2,000万元&不存在未弥补亏损
      4. **发行后** 股本总额不少于3,000万元
      5. 注册资本足额缴纳，主要资产无重大权属纠纷
      6. 应主要经营一种业务，经营合法
      7. 最近2年主营业务、董&高没有重大变化；实际控制人未变更
      8. 股权清晰，控股股东无重大权属纠纷
      9. 公司治理结构完善，内部控制健全有效
      10. 会计基础工作规范，无保留意见审计报告
      11. 董监高具备法定资格(不具有5种公司法禁止情形)，且不存在下列情形(x3)：
        - 被证监会采取证券市场禁入措施尚在禁入期的
        - 最近3年受到证监会行政处罚 or 1年内受到证交所公开谴责
        - 涉嫌犯罪/违法违规被立案侦查/调查，尚未有明确结论意见
      12. 发行人/控股股东/实际控制人近3年：
        - 无重大违法行为
        - 无未经核准擅自公开/变相公开发行债券
        - 违法行为发生于3年前但目前仍持续
  2. 股票承销：
    + 包销/代销：证券公司应当保证将证券先行出售给认购人，不得预先购入/留存
    + 向不特定对象公开发行的证券票面总值超过5,000万元的，应当由承销团承销
    + 承销期限最长不得超过90日
    + 采用代销方式，代销期限届满，售出的数量未达拟公开发行数量70%的，为发行失败
- 上市公司增发股票：非公开/公开；公开增发股票分配股(向原股东配售)/增发(向不特定对象公开募集)
  1. 条件：
    + 上市公司公开发行证券的一般条件(x10)：证券指股票(配股/增发) + 可转债(一般/可分离)
      1. [董监高] 董监高具备任职资格，无违反公司法一般义务(147)/禁止行为(148)情形；且无3年内受到证监会行政处罚 or 1年内受到证交所公开谴责情形
      2. [股东&担保] 公司与控股股东/实际控制人不存在人格混同，最近1年内无违规对外担保
      3. [净利润] 最近3个会计年度连续盈利，净利润(>0)取扣除非经常性损益前后孰低者
      4. [业务稳定] 高级管理人员/核心技术人员稳定，最近1年内无重大不利变化
      5. [发债历史] 最近2年内曾公开发行证券的，不存在发行当年营业利润比上年下降50%以上情形
      6. [审计报告] 最近3年及1期财务报告审计结果：
        - 不允许保留意见/否定意见/无法表示意见
        - 带强调事项段的无保留意见，须涉及事项无/发行前已消除重大不利影响
      7. [现金股利] 最近3年以现金方式累计分配的利润不少于(≥) 最近3年实现的年均可分配利润的30% (Σ/μ ≥ 30%)
      8. [禁止财务性投资] 除金融类企业，资金使用项目不得为财务性投资（交易性/可供出售金融资产/贷出/委托理财）
      9. [不损害投资者权益/公共利益] 上市公司不存在下列行为(x5+1)：
        - 发行申请虚假陈述
        - 擅改前次公开发行证券募资用途未纠正
        - 公司最近1年内受到过证交所公开谴责
        - 公司/控股股东/实际控制人最近1年存在未履行公开承诺的行为
        - 公司&董&高涉嫌犯罪/违法违规被立案侦查/调查，尚未有明确结论意见
        - 其他严重损害投资者合法权益/社会公共利益的情形
      10. [特别决议] 上市公司发行证券须经股东大会回避+特别决议通过
    + 配股：
      1. 附加条件(+3)：
        - 拟配售股份数量不超过本次配售前股本总额的30%
        - 控股股东应在股东大会召开前公开承诺认配股份的数量
        - 采用代销发行（不能包销）
      2. 发行失败(x2)，加算利息返还：
        - 控股股东不履行认配承诺
        - 代销期限届满，认购未达计划70%
    + 增发的附加条件(+3)：向不特定对象公开募集
      1. 最近3个会计年度加权平均净资产收益率平均不低于6%(ROE≥6%)，计算时净利润取孰低者(简易ROE = 净利润/净资产)
      2. 除金融类企业外，最近一期期末不存在持有金额较大的财务性投资
      3. 发行价格应不低于(≥) 公告招股意向书 前20个交易日公司股票均价 **或** 前1个交易日均价 (P ≥ μ<sub>t-20</sub> **or** μ<sub>t-1</sub>)
    + 非公开发行股票的条件
      1. 发行对象和认购条件：
        - 发行对象应符合股东大会决议规定的条件，且 **不超过10名**（同一基金公司管理的基金们算一个；信托公司只能用自有资金认购；境外战略投资者要报批）
        - 不同类型发行对象的定价原则和锁定期：
          1. 定价原则由董事会决议后报经股东大会批准；认购的股份3年内不得转让：
            + 控股股东/实际控制人/受其控制的关联人
            + 通过认购本次发行的股份取得实际控制权者
            + 拟引入的境内外战略投资者
          2. 以上3种之外的发行对象，以竞价方式确定价格和发行对象；认购的股份1年内不得转让
        - 定价总原则：
          1. 发行价格不低于定价基准日前20个交易日公司股票均价的90% (P ≥ 90% · μ<sub>t-20</sub>)
          2. 定价基准日(3选1)：本次非公开发行的董事会决议公告日/股东大会决议公告日/发行期首日
          3. μ<sub>t-20</sub> = 该20日股票交易总额 / 总量
      2. 法定障碍(x6+1)：
        - 本次申请文件虚假陈述
        - 上市公司权益被控股股东/实际控制人严重损害且尚未消除
        - 上市公司及其附属公司违规对外提供担保尚未解除
        - 现任董&高3年内受到证监会行政处罚 or 1年内受到证交所公开谴责 (公开增发这里是董监高)
        - 公司/董/高涉嫌犯罪/违法正被立案侦查/调查
        - (\*)[非标准报告留了口子] 最近1年及1期财务报表被出具保留/否定/无法表示意见；所涉及事项的重大影响已消除或本次发行涉及重大重组的除外
        - 其他
  2. 程序：
    + 一般程序：公开/非公开
      1. 董事会决议
      2. 股东大会批准：回避 + 出席 + ≥2/3
      3. 保荐
      4. 审核
      5. 发行：核准后6个月内
      6. 承销
    + 非公开发行的特别规定：非公开发行对象均属原前10名股东的，可由上市公司自行销售(不必承销)
- 优先股发行和交易：
  1. 发行人范围：
    + 公开发行优先股：上市公司
    + 非公开发行优先股：上市公司&非上市公众公司
  2. 发行条件：
    + 已发行的优先股不得超过公司普通股总数的50%
    + 筹资金额不得超过发行前净资产的50%
  3. 公开发行的特殊要求(x4)：
    + 采用固定股息率
    + 有可分配税后利润的情况下必须向优先股股东分配
    + 未足额分配的累积到下一年
    + 优先股股东分配后不再和普通股股东一同参加剩余利润分配

### 第三节 公司债券的发行与交易
- 普通债券的发行：
  1. 一般规定
    + 股东大会决议：一般决议
    + 公开/非公开均可
    + 期限1年以上，每张面值100元
  2. 公开发行：应经证监会核准
    + 一般条件(x6)：
      1. 股份有限公司净资产不低于3,000万，有限责任公司净资产不低于6,000万
      2. 发行后累计债券余额不超过最近一期期末净资产40%
      3. 业务/资金投向合规
      4. 最近3个会计年度实现的年均可分配利润不少于1年的利息
      5. 利率不超限
      6. 内控健全有效
    + 发行对象：公众投资者/合格投资者
      1. 可向公众投资者公开发行的额外条件(x3+1, 满足的也可以选择只向合格投资者公开发行)
        - 最近3年无债务违约/迟延支付本息
        - 最近3个会计年度年均可分配利润不少于1年利息的1.5倍
        - 债券评级AAA级
        - 其他
      2. 未达上述标准的只能面向合格投资者公开发行
      3. 仅向合格投资者公开发行的，简化核准程序
    + 法定障碍(x5+1)：
      1. 前一次公开发行的公司债券尚未募足
      2. 对债券/其他债务违约/迟延支付本息，仍处于继续状态
      3. 违规改变公开发行债券的资金用途
      4. 最近3年公司报表虚假陈述/有其他重大违法行为
      5. 本次申请文件虚假陈述
      6. 其他
    + 募集资金必须用于核准的用途，不得用于弥补亏损和非生产性支出
    + 可以一次核准，分期发行
      1. 首期1年内；剩余2年内发完
      2. 募集说明书有效期6个月，及时更新
      3. 每期发行完5个工作日内报备
    + 合格投资者(x6+1+1, 含三类自然人)：穿透核查，合并计算人数（理财产品的投资者/合伙企业的合伙人等）
      1. 金融机构、登记的私募基金管理人
      2. 金融机构的理财产品
      3. 净资产不低于1,000万元的企事业单位法人/合伙企业
      4. QFII, RQFII
      5. 养老基金/社会公益基金
      6. 名下金融资产不低于300万元的个人投资者
      7. 发行人的董监高&持股比例超过5%股东 → 只在非公开时可以无视合格投资者条件
      8. 其他
  3. 非公开发行：
    + 应当向合格投资者发行；不得采用公开方式(x3)宣传；每次发行对象不超过200人
    + 可不进行信用评级，发行完5个工作日内报备，**无需核准**
    + 仅限于合格投资者范围内转让，持有同次债券者合计不超过200人
  4. 信息披露：重大事项(x12+1)
    + 经营方针/范围/外部条件重大变化
    + 债券评级变化
    + 主要资产被查封/扣押/冻结
    + 发生债务违约
    + (\*)当年累计新增借款/对外担保超过上年末净资产20%
    + (\*)放弃债权/财产超过上年末净资产10%
    + (\*)发生超过上年末净资产10%的重大损失
    + 发行人减资/合并/分立/解散/申请破产
    + 重大诉讼/仲裁/行政处罚
    + 偿债保障措施(保证/担保等)重大变化
    + 发行人重大变化导致可能不符债券上市条件
    + 发行人/董监高涉嫌犯罪
    + 其他
  5. 债券持有人权益保护(x4)
    + 信用评级
    + 受托管理人：上市公司必须聘请
    + 债券持有人会议：受托管理人召集会议情形(x9+1)；受托管理人撂挑子，单独/合计10%以上持有人可自行召集
      1. 变更募集说明书
      2. 修改会议规则
      3. 变更受托管理人/协议
      4. (\*)发行人不能按期支付本息
      5. (\*)发行人减资/合并/分立/解散/申请破产
      6. (\*)偿债保障措施(保证/担保等)重大变化
      7. 发行人、单独/合计持有本期10%以上者书面提议召开
      8. 管理层不能正常履职
      9. 发行人提出债务重组方案
      10. 其他
    + 担保：第三方担保/商业保险/抵押质押/限制发行人行为/设置回售条款等
- 上市公司公开发行可转换公司债券：一般/可分离
  1. 满足上市公司公开发行证券的一般条件(x10)
  2. 发行一般可转换的额外条件(10+3)：可分离全部继承
    + 最近3个会计年度加权平均净资产收益率平均不低于6%(ROE≥6%)，计算时净利润取孰低者
    + 发行后累计债券余额不超过最近一期期末净资产40%
    + 最近3个会计年度年均可分配利润不少于1年的利息
  3. 发行分离交易的可转换公司债券的额外条件(10+4)：纯债和权证捆绑发行、分别交易，不要求同步操作
    + 最近一期期末经审计净资产不低于15亿元
    + 最近3个会计年度年均可分配利润不少于1年的利息
    + 最近3个会计年度 A.经营活动产生的现金流量净额均值不少于1年的利息 or B.加权平均净资产收益率平均不低于6%(ROE≥6%)，计算时净利润取孰低者
    + 发行后累计债券余额不超过最近一期期末净资产40%；所附权证全部行权后募集资金总量不超过纯债部分的金额
  4. 期限[1, 6]年，每张面值100元
  5. 持有人权利保护：公开发行可转债(一般&可分离)应当提供担保，提供的担保应当为全额担保/连带责任担保；最近一期期末经审计净资产不低于15亿者可不提供担保
  6. 债转股：
    + 发行结束之日起6个月后方可
    + 转股价格应不低于募集说明书公告日 前20个交易日公司股票均价 **和** 前1个交易日均价 (P ≥ μ<sub>t-20</sub> **and** μ<sub>t-1</sub>)
  7. 信息披露：与公开增发股票基本相同，主要不同在募集说明书
    + 可约定赎回条款
    + 可约定回售条款
    + 应当约定改变资金用途时债券持有人可回售
    + 应当约定转股价格调整的原则及方式
    + 因配股/增发/送股/派息/分立等原因引起股份变动的，应当同时调整转股价格
    + 约定转股价格向下修正条款的(损害已有股东利益)，应同时约定：
      1. 价格修正方案须经股东大会回避+特别决议通过（持有可转债的回避+出席+≥2/3）
      2. 修正后的转股价格不低于该股东大会召开之日 前20个交易日公司股票均价 **和** 前1个交易日均价 (P ≥ μ<sub>t-20</sub> **and** μ<sub>t-1</sub>)
- 交易：
  1. 上市交易的条件(x3, 普通&可转换)：
    + 期限1年以上
    + 实际发行额不少于5,000万元
    + 符合法定债券发行条件
  2. 暂停上市情形(x5)：
    + 重大违法
    + 不符合上市交易条件
    + 募集资金未按核准用途使用
    + 未履行约定义务
    + 最近2年连续亏损
  3. 终止上市：
    + 上述1/4两项后果严重的
    + 上述2/3/5项限期未能消除的
    + 公司解散/被宣告破产
  4. 分离交易的可转换公司债券的交易：
    + 认股权证的行权价格应不低于募集说明书公告日 前20个交易日公司股票均价 **和** 前1个交易日均价 (P ≥ μ<sub>t-20</sub> **and** μ<sub>t-1</sub>)
    + 认股权证存续期间不超过纯债部分的期限，自发行结束之日起不少于6个月；存续期限 **不得调整**
    + 行权期间为[发行6个月后, 存续期届满前]，或约定的子集

### 第四节 股票的上市与交易
- 上交所交易时间：非节假日的周一至周五
  1. 开盘集合竞价：09:15 - 09:25
  2. 连续竞价：09:30 - 11:30; 13:00 - 15:00
  3. 大宗交易：15:00 - 15:30
- 股票上市条件(x4)：股份有限公司
  1. 已经过证监会核准
  2. 股本总额不少于3,000万元
  3. 公开发行的股份比例下限：
    + 股本总额 [3千万, 4亿]：25%以上
    + 股本总额 (4亿, +∞)：10%以上
  4. 最近3年无重大违法行为，财务报告无虚假陈述
- 股票退市：主动/强制
  1. 主动退市：
    + 主动申请退市/转市：股东大会决议（**两个出席+2/3**），决议后15日内交申请，提交后5日内决定受理，受理的提交后15日内做决定
      1. [总数2/3] 出席会议的股东所持表决权2/3以上通过
      2. [小股东2/3] 出席会议的2种股东之外的其他股东所持表决权2/3以上通过：
        - 上市公司的董监高
        - 单独/合计持有5%以上股份的股东
    + 通过要约收购/合并/解散实施退市：证交所15日内作出终止上市决定
  2. 重大违法行为强制退市：欺诈发行、重大信息披露违法
    + 对欺诈发行/重大信息披露违法公司实施暂停上市
    + 1年内实施终止上市；例外：撤销处罚/不予立案/无罪判决等 or 全面纠正违法行为的，可申请恢复上市
  3. 不满足交易标准的强制退市：股权分布不具备上市条件
    + 社会公众股东持有的股份连续20个交易日低于比例下限（[3kw, 400m]25%/(400m, +∞)10%）
    + 社会公众股东指不包括以下股东(x2)的其他股东：
      1. 持股10%以上股东及其一致行动人
      2. 董监高及其关联人
  4. 退市整理期：30个交易日
  5. 退市后：
    + 主动退市的：证交所转让或其他安排
    + 强制退市的：新三板挂牌转让

### 第五节 上市公司收购和重组
- 具有实际控制权的情形(x4+1)：收购的目的，不能达到实际控制目的而受让股票的行为不称为收购
  1. 持股50%以上
  2. 可实际支配表决权超过30%
  3. 能够决定董事会半数以上成员选任
  4. 足以对股东大会决议产生重大影响
  5. 其他
- 收购人：意图取得实际控制权的投资者及其一致行动人
  1. 一致行动人情形(x11+1)：无相反证据即可认定；所持股份合并计算
    + 相互之间有股权控制关系
    + 受同一主体控制
    + 共用董监高
    + 有参股且能对被参股公司重大决策产生重大影响
    + 银行以外法人/组织/自然人为投资者取得股份提供融资安排
    + 存在合伙/合作/联营等经济利益关系
    + 持有投资者30%以上股份的自然人 with 投资者
    + 在投资者任职的董监高 with 投资者
    + 前两项人员(30%/董监高)的亲属（父母配偶子女、兄弟姐妹及其配偶、配偶的父母&兄弟姐妹&其配偶、子女的配偶）with 投资者
    + 上市公司董监高/其前项所述亲属/董监高&亲属直接or间接控制的企业，相互之间同时持有本公司股份
    + 上市公司董监高&员工 with 其所控制/委托的法人/其他组织 同时持有本公司股份
    + 其他关联关系
  2. 禁止投资者收购上市公司情形(x4+1)：
    + 数额较大债务到期未清偿，且处于持续状态
    + 最近3年有/涉嫌重大违法行为
    + 最近3年有严重证券市场失信行为
    + 自然人投资者存在不得担任董监高的5种情形（行为能力/黑五类5年/破产3年/公司违法3年/欠大钱不还）
    + 其他
  3. 锁定义务：
    + 收购完成后1年内不得转让
    + 同一控制人控制下的主体之间转让不受此限
    + 通过集中竞价增持的，最后一笔增持股份登记过户时视为收购完成
  4. 收购的支付方式：现金/可转让证券/其他
- 持股权益披露：
  1. 通过证券交易所交易，投资者与一致行动人共同持有股份达5%时，应在3日内书面报告&通知上市公司公告&期间内不得买卖该公司股票
  2. 达5%后，每增加/减少5%（优先股不计入），依前款报告/公告，报告期间内和其后2日内(共5日)不得买卖
  3. 通过协议转让则不严格要求恰好5%，达到/超过5%后依前款报告即可；后续报告要求在5%整数倍上(10%, 15%等)
- 要约收购程序：自愿/强制；预定收购股份比例不得低于已发行股份5%
  1. 提示性公告：
    + 编制要约收购报告书
    + 对报告书摘要作提示性公告
  2. 要约公告和竞争要约：届满前15日
    + 提示性公告日起60日内未公告收购报告书的，应于次一工作日公告，此后每30日公告一次，直至公告报告书
    + 公告报告书前可自由取消计划，公告原因，1年内不得卷土重来
    + 收购期限[30, 60]日，出现竞争要约除外
    + 收购要约确定的承诺期内不得撤销收购要约
    + 承诺期内变更要约必须及时公告；期限届满前15日内不得变更，出现竞争要约除外
    + 出现竞争要约：
      1. 初始要约不足15日的，应延长收购期限，延长后应不少于15日 & 不超过最后一个竞争要约期满日
      2. 竞争要约发出人最迟必须在初始要约期满前15日发出要约收购提示性公告
  3. 要约对象和条件：同股同权
    + 对同种股票要约价格不得低于 提示性公告日前6个月内收购人取得该种股票所支付的 **最高价格**
    + 收购条件应适用于所有股东；对不同种(普通/优先)股东可提出不同条件
  4. 收购期限内不得通过其他方式获得股票
    + 不得卖出
    + 不得采取要约规定之外的形式/超出要约的条件 购买被收购公司的股票
  5. 被收购公司董事会义务：
    + 要约收购期间，不得进行未批准重大经营调整/资产处置
    + 董事不得辞职
  6. 预受要约：
    + 收购期内股东的同意视为预受
    + 期限届满3个交易日前可以撤回，之后不得撤回
  7. 要约期满：
    + 部分要约(向被收购公司 **所有股东** 发出收购其部分股份的要约)：预受超过预定数量，按同等比例收购
    + 全面要约(向被收购公司所有股东发出收购其全部股份的要约)：购买预受的全部股份
    + 3个交易日内结算/过户
    + 15日内提交书面报告&公告
- 强制要约制度：
  1. 投资者取得上市公司已发行股份=30%时(停下)，继续进行收购的，必须通过要约方式，全面要约or部分要约：
    + 不管什么方式(证券市场购买/协议转让)，只要在=30%停下了，接下来可在全面要约/部分要约中选择
    + 通过协议转让且有>30%的部分，履行协议前必须发出全面要约
    + 满足一定条件可向证监会申请豁免/简易申请/免于申请直接过户
    + 不予豁免的，通知之日起30日内减持至≤30%以下，也可避免强制全面要约义务
  2. **豁免事项**(x3类, 2+2+7)：
    + 可提出免于以要约方式增持股份申请的情形(x2+1)：受理后20个工作日内答复决定；无实质变化+做好事
      1. [实际控制人未变] 收购人与出让人能够证明属同一控制人控制，本次转让未导致实际控制人变化
      2. [挽救重组+3年不卖] 上市公司面临严重财务困难，收购人提出的重组方案股东大会已通过 & 收购人承诺3年内不转让该公司股份
      3. 其他
    + 适用简易申请的情形(x2+1)：收到后10个工作日内无异议即可过户；被动超限
      1. 经批准进行国有资产无偿划转/变更/合并，导致投资者持股超过30%
      2. 因按股东大会批准向特定股东回购股份而减少股本，使投资者持股超过30%
      3. 其他
    + 免于提出申请直接办理转让过户的情形(x7)：完成后3日内公告 *背诵*
      1. 经股东大会非关联股东批准(回避+出席+≥2/3)，投资者取得向其发行的新股导致持股超过30%；承诺3年内不转让本次获得的新股，且股东大会同意其免于全面要约义务
      2. 已经持股≥30%以上，自上述事实发生之日起 **1年后**，每12个月内增持不超过≤2%的股份
        - 该项增持≤2%股份锁定期(不得转让)为增持完成之日起6个月
        - 通过集中竞价增持累计1%的，发生之日通知&次日公告
      3. 已经持股≥50%以上，继续增持不影响公司的上市地位(公众持股25%/10%)
        - 通过集中竞价增持累计1%的，发生之日通知&次日公告
        - 通过集中竞价增持累计2%的，发生之日&公告之日均不得再行增持
      4. 金融机构开展承销/贷款等业务导致持股超过30%，没有实际控制公司的行为/意图，且已提出合理期限内向非关联方转让的解决方案
      5. 因继承导致持股超过30%
      6. 因履行约定购回式证券交易协议导致持股超过30%，且能够证明标的股份表决权在协议期间未发生转移
      7. 因所持优先股表决权恢复导致持股超过30%
- 特殊类型收购：协议收购；间接收购不讲
  1. 过渡期内 **禁止事项**：协议签订至股份过户完成期间为过渡期
    + 收购人不得提议改选董事会；确有充分理由的，来自收购人的董事不得超过≤1/3
    + 被收购公司不得为收购人及其关联方提供担保
    + 被收购公司不得增发/重大资产买卖/重大投资/与收购人关联交易；收购人挽救行为除外
  2. 股权过户：
    + 收购报告书公告后30日内未过户应立即公告
    + 仍未过户的期间内每隔30日公告进展
  3. 管理层收购条件(x5)：3个1/2+1个2/3
    + 独立董事比例≥1/2以上
    + 董事会非关联董事通过(回避+全体+>1/2)
    + ≥2/3以上独立董事同意
    + 股东大会一般决议通过(回避+出席+>1/2)
    + 董监高不存在公司法148条规定情形(挪用公司资金/与公司交易/经营竞业等)；最近3年无证券市场不良诚信记录
- **上市公司重大资产重组**：
  1. 界定：
    + 通过经营之外的买卖重大资产/其他资产交易使得主营业务/资产/收入重大变化
    + 其他资产交易包括(x3+1)：
      1. 新设企业并对已有企业增减资
      2. 受托经营/租赁其他企业资产 or 将经营性资产委托他人经营/租赁
      3. 接受附义务的资产赠与/对外捐赠
      4. 其他
    + 经核准发行证券，按披露用途使用资金购买资产/对外投资的，不属于重大资产重组
    + 上市公司及其控股公司购买/出售资产达下列标准(x3)之一，构成重大资产重组：50%+总资产/营业收入/净资产
      1. 总额占最近1个会计年度经审计合并报告期末总资产50%以上
      2. 涉及的资产在最近1个会计年度产生的营业收入占同期经审计合并报告营业收入50%以上
      3. 净额占最近1个会计年度经审计合并报告期末净资产50%以上，**且** 超过5,000万元
    + 计算比例注意事项：资产额“孰高者” + 连续交易“累计数”
      1. 购入股权：
        - 对被投资企业控股权未变动
          1. 资产总额：被投资企业资产总额 x 股权比例 & 成交金额 孰高者
          2. 营业收入：被投资企业营业收入 x 股权比例
          3. 资产净额：被投资企业资产净额 x 股权比例 & 成交金额 孰高者
        - 取得被投资企业控股权
          1. 资产总额：被投资企业资产总额 & 成交金额 孰高者
          2. 营业收入：被投资企业营业收入
          3. 资产净额：被投资企业资产净额 & 成交金额 孰高者
      2. 出售股权：
        - 控股权未变动：全部为被投资企业相应指标的相应股权比例
        - 丧失控股权：全部为被投资企业相应指标
      3. 非股权资产：
        - 购买：账面值 & 成交金额 孰高者
        - 出售：账面值
        - 不涉及负债的：不适用净额标准
      4. 同时购买&出售：分别计算比例，取孰高者
      5. 1年内连续购买/出售，计算累计数
  2. 要求：
    + 一般要求(x7)：
      1. 符合法律法规
      2. 不使上市公司丧失上市条件
      3. 定价公允
      4. 权属清晰，权属转移无障碍，相关处理合法
      5. (\*)有利于增强持续经营能力，不存在重组后主要资产为现金/无具体业务情形
      6. 符合公司独立性要求
      7. 法人治理结构有效
    + 借壳上市的特殊要求：与公开发行上市标准相同；创业板不得借壳
      1. 自控制权变更之日起，上市公司向收购人及其关联人购买的资产总额，占上市公司控制权变更前1个会计年度经审计合并报告期末资产总额100%以上的，称为借壳上市
      2. 主板&中小板上市公司购买资产对应的经营实体应当是有限责任/股份有限公司，且符合IPO条件；创业板公司 **不得** 被借壳
  3. 发行股份购买资产：
    + 规定(x4+1)：审计/违法限制较松
      1. 充分披露说明好处（经营/财务状况/独立性等）
      2. 最近1年及1期财报无保留意见 or 保留/否定/无法表示意见的，所涉及事项重大影响已经/将被本次交易消除
      3. 公司/董/高不存在违法正被调查情形 or 该行为已终止满3年&交易方案有助消除不良后果&不影响追责
      4. 充分披露说明是好资产且能按时买到
      5. 其他
    + 控制权不变情形下，可向控股股东/实际控制人/其关联人之外的特定对象(即非公开)发行股份购买资产；特定对象认购后，上市公司用同一次募集的资金向该特定对象购买资产的，视同发行股份购买资产
    + 发行价格不得低于市场参考价的90%；市场参考价为本次董事会决议公告日 前20/60/120个交易日股票均价(3选1)(P ≥ 90% · μ<sub>t-20</sub> or μ<sub>t-60</sub> or μ<sub>t-120</sub>)
    + 锁定期：1年/3年
      1. 以资产认购取得股份，1年内不得转让
      2. 有以下情形(x3)之一，3年内不得转让：
        - 购买人是控股股东/实际控制人/其关联人
        - 通过认购本次股份取得实际控制权
        - (\*)取得本次股份时，对用于认购的资产持续拥有权益的时间不足1年
  4. 公司决议：
    + 重大资产重组属于股东大会特别决议，回避+出席+≥2/3
    + 相应股东大会应为现场会议+网络投票
    + 董监高 & 单独/合计持股5%以上股东以外的股东的投票情况，应单独统计并披露
  5. 证监会核准：仅借壳上市 & 发行股份购买资产需要核准

### 第六节 证券欺诈的法律责任
- 虚假陈述行为：
  1. 行为界定(x4种)：
    + [不正当披露] 未按规定的期限/方式等披露，构成 未按照规定披露信息
    + [虚假记载] 记载与真实情况不符，构成 虚假记载
    + [误导性陈述] 信息披露不完整不准确，可能致使错误判断，构成 误导性陈述
    + [重大遗漏] 未按规定要求披露，遗漏重大事项，构成 重大遗漏
  2. 法律责任
    + **行政责任**：机构/个人
      1. 发行人/上市公司等信息披露义务人(法人)：责令改正，给予警告，罚款 [30w, 60w]
      2. 直接负责主管人员/直接责任人员：给予警告，罚款 [3w, 30w]；有责任自然人类型：
        - 董监高
        - 有直接因果关系的其他人员
        - 指使虚假陈述的控股股东/实际控制人(或法人的负责人)
      3. 证券服务机构：律师/会计师/资产评估师等
        - 责令改正，没收业务收入，暂停/撤销服务许可，罚款：业务收入[1倍, 5倍]
        - 直接负责主管人员/直接责任人员：警告，撤销证券从业资格，罚款 [3w, 10w]
      4. 从轻或减轻处罚的考虑情形(x5+1)：
        - 未直接参与
        - 违法行为被发现前，及时主动要求公司纠正或向监管机构报告
        - 获悉违法后，向上级提出质疑并采取适当措施
        - 配合监管机构调查且有立功表现
        - 受胁迫参与
        - 其他
      5. 不予处罚的考虑情形(x3+1)：
        - 对违法事项提出具体异议并记载于会议记录等，且投反对票的
        - 涉及期间因不可抗力/失去人身自由等无法正常履行职责的
        - 不负有主要责任人员发生后及时向公司和监管机构报告的
        - 其他
      6. **不得单独作为** 不予处罚的情形(x5)：
        - 不直接从事经营管理
        - 能力不足/无相关职业背景
        - 任职时间短/不了解情况
        - 相信专业机构/人员的意见/报告
        - 受到股东/实际控制人控制或其他外部干预
      7. 从重处罚情形(x4+1)：
        - 不配合/拒绝/阻碍/暴力or威胁干扰执法
        - 变造/隐瞒/毁灭证据，提供伪证
        - ≥2次以上违反信息披露规定受到处罚/处分
        - 有信息披露不良诚信记录
        - 其他
    + 刑事责任：欺诈发行股票、债券罪；违规披露、不披露重要信息罪
    + 民事责任：
      1. 责任分类：全部连带
        - 无过错责任(必须承担责任)：发行人/上市公司
        - 过错推定责任(x3类, 可自证无过错的免责)：
          1. 发行人/上市公司的董监高/其他直接责任人
          2. 保荐人/承销商
          3. 证券服务机构
        - 过错责任(举证责任在原告)：控股股东/实际控制人
      2. 责任追究难题解决：确定因果关系 & 诉讼方式(没讲)
        - 因果关系认定采取推定方式：
          1. 确定两个时间点：
            + 虚假陈述实施日(A)：披露日(积极)；法定期限最后一日(消极)
            + 虚假陈述揭露日/更正日(B)：立案稽查消息发布日/导致停牌的媒体揭露日
          2. 确定买入&损失产生时间段：符合要求即推定有因果关系
            + [A, B) 买入
            + [B, +∞) 因卖出/持续持有该证券产生亏损
      3. 赔偿责任大小的确定：基准日/填坑原则
        - 损失计算基准日(C)：
          1. B日起，累计成交量达可流通部分100%之日，不计算大宗交易/协议转让
          2. 开庭审理前未达前项规定之日，为B日后第30个交易日(6周)
          3. 已退出/停止交易的，为摘牌/停牌前一日；恢复交易的以第1项规则确定
        - 以实际发生的损失为限：投资差额损失 + 损失部分的佣金&印花税
          1. [B, C] 卖出：(买入均价 - 卖出均价) x 数量
          2. (C, +∞) 卖出/仍持有：(买入均价 - [B, C]之间每个交易日收盘价均值) x 数量
          3. 投资人以股东身份取得的收益(红利/红股/配股/增发股等)，**不得冲抵** 赔偿金额
          4. 已除权的证券，价格/数量应复权计算；除息没讲
- **内幕交易行为**：
  1. 内幕信息(x20+1+6+1)：
    + 应报送临时报告的重大事件(x20+1)
    + 分配股利/增资计划
    + 股权结构重大变化
    + 债务担保重大变更
    + 营业用主要资产抵押/出售/报废，一次超过该资产>30%
    + 董监高可能承担重大损害赔偿责任
    + 上市公司收购方案
    + 其他
  2. 内幕交易行为认定：
    + 在内幕信息敏感期，知情人员&非法获取者，买卖/泄露/建议他人买卖该证券，即构成内幕交易
    + 内幕信息敏感期：内幕信息产生至公开之间
    + 内幕信息知情人员(x6+1)：自然人
      1. 董监高
      2. 实际控制人/持股5%以上股东及其董监高
      3. 被控股公司及其董监高
      4. 因所任职务之便可以获取内幕信息的人员
      5. 监管机构工作人员/其他法定管理人员
      6. 保荐人/承销商/证交所/证券登记结算机构/证券服务机构的有关人员
      7. 其他
    + 非法获取证券内幕信息的人员：
      1. 窃取骗取等非法手段
      2. 知情人近亲属明示暗示
      3. 相关交易明显异常/无正当理由/无正当信息来源的
  3. 责任推定：
    + 主体适格：知情人/其配偶父母子女/与其联络接触者/非法获取人
    + 时间吻合：证券交易活动与内幕信息吻合
    + 当事人可举证推翻：合理解释/提供证据排除
  4. 不属于(刑法上)内幕交易的情况(x4)：
    + 单独/合计持有5%以上股份者进行收购的
    + 按照事先订立的书面合同/指令/计划交易的
    + 依据已被他人披露的信息交易的
    + 其他正当理由/正当信息来源
  5. 短线交易：6个月
    + 主体：上市公司董监高 & 5%以上股东
    + 行为：将该公司股票 买入后6个月内卖出 or 卖出后6个月内买入；最后一笔买/卖时点起算6个月和股份数
    + 结果：董事会应将收益收归公司所有
    + 例外：证券公司因包销购入售后剩余股票成为5%以上股东的，卖出不受此限
    + 救济：董事会不执行短线交易归入权，其他股东有权要求在30日内执行；仍不执行的，股东代表诉讼
    + 举例：02/01买入5w股，02/10买入10w股，08/02全部卖出；以02/10起算6m，以10w股算短线交易利润
- 操纵市场行为：*了解为主*
  1. 利用资金/持股/信息优势连续买卖
  2. 自己/串通者之间交易(虚假交易)
  3. 其他手段：修改系统数据；咨询报告/荐股；编造/传播虚假信息等

## 第八章 企业破产法律制度
with 物权/合同/公司法；须准确理解；小题少+必1道综合题

### 第一节 破产法律制度概述
- 强制清算财产，公平有序地清偿全体债权人；广义上包括破产清算、重整(挽救一段时间)、和解(打折)

### 第二节 破产申请与受理
- 破产原因：概括主义
  1. 不能清偿到期债务，并且资产不足以清偿全部债务/明显缺乏清偿能力
  2. **债务人本人** 不能清偿到期债务即为发生破产原因，其他人的连带责任/担保责任不能视为清偿能力或其延伸
  3. 账面资产虽大于负债，但存在下列情形(x4+1)之一，认定明显缺乏清偿能力：
    + 流动资金不足但财产不能变现/变现即意味着失去经营条件
    + 法定代表人下落不明且无人管理财产
    + 经强制执行仍无法清偿债务：任何一个债权人经强制执行未得清偿，每个债权人 **均有权** 提出破产申请
    + 长期亏损且扭亏困难
    + 其他
- 破产申请的提出：同时适用破产清算/和解/重整，和解/重整申请也有一些特殊规定
  1. 申请人：
    + 债务人：重整、和解、破产清算
    + 债权人：重整、破产清算
      1. 对担保债权人没有限制
      2. 破产企业的职工提出申请：要求职工(代表)大会决议通过
    + 税务机关/社保机构：破产清算
  2. 破产案件管辖：债务人住所地人民法院
    + 主要办事机构所在地 → 注册登记地
    + 破产申请提交后受理前，申请人可以请求撤回
- 破产申请的受理：程序(x) & **效力**
  1. 债务人抗辩异议：提出不存在破产原因；债权有无(√)/部分数额争议(x)/担保有无(x)
    + 提出有清偿能力/资产超过负债，但不能立即清偿/达成和解的：异议不成立
    + 是否享有债权：需要合理证据/理由，无债权不能申请破产
    + 承认债权，对债权的具体数额有争议：不影响破产原因成立
    + 承认债权，对有无担保有异议：不影响破产原因成立
  2. 受理程序(x)：
    + 需要补正材料，一次性告知
    + 法院收到后通知债务人，债务人可以提出异议
    + 异议期满裁定是否受理：
      1. 受理：送达申请人/债务人；债务人提交财产相关材料
      2. 不受理：送达申请人，可以提上诉
    + (\*)法院受理后→宣告前，发现 **受理时** 未发生破产原因，可以驳回申请：
      1. 受理后破产原因消失，不得驳回
      2. 对驳回裁定不服可以上诉
    + 裁定受理破产申请，应同时指定管理人，25日内通知债权人并公告
  3. **破产申请受理的法律效力**：
    + 债务人有关人员的义务(x5)：指法定代表人，也可经法院决定包括财务管理人员等
      1. 妥善保管财产/账簿
      2. 根据法院/管理人的要求工作，如实回答询问
      3. 列席债权人会议并如实回答询问
      4. 未经许可不得离开住所地
      5. 不得新任董监高
    + 对个别债务人清偿：
      1. 受理后个别清偿无效
      2. 对有物权担保的债权人，在担保物市场价值内清偿不受限制
    + 债务人的债务人/财产持有人清偿债务/交付财产：
      1. 向管理人
      2. 故意违反规定向债务人交付财产，使债权人受损失，不能免除责任（以债权人因此受到的损失范围为限，无损失无责任）
    + 受理前成立的 + 双方均未履行完毕的合同：
      1. 管理人有权决定解除 or 继续履行，并通知对方当事人：保障债权人权益最大化为原则
      2. 自受理之日起2个月内未通知对方当事人/催告后30日未答复，视为解除合同/丧失要求继续履行的选择权（仅是对选择履行权的限制，若此后双方均同意继续履行也可）
      3. 管理人要求继续履行，对方无权拒绝，但有权要求提供担保；不提供担保的视为解除合同
      4. 选择履行权只能行使一次，不能反向或多次行使；明示默示行使均视为已行使
      5. 破产法赋予的选择履行权行使完毕，不妨碍其他法律赋予的/合同约定的解除权/另行签订新合同
      4. 特殊种类合同(x4)下选择履行权受限：
        - 破产企业为他人提供担保的合同：无权解除
        - 尚未履行完毕的保险合同：无权解除
        - 金融衍生品交易合同：必须解除
        - 对外出租不动产合同：不得任意单方解除；可带租约出售，承租人优先购买权
    + 债务人财产的保全措施和执行程序：
      1. (已有的)保全措施应当解除；受理法院可以按需采取新的保全措施
      2. 已作出生效民事判决书/调解书尚未执行完毕的，执行程序应当终止(暂停 pause)；债权人依判决书申报债权
        - 对有物权担保的债权，在担保财产市场价值内可以继续执行
        - 若宣告破产前裁定驳回申请，原顺位恢复保全/继续执行
        - 执行程序未终止的，应依法纠正；依法执行回转的财产为债务人财产
    + 债务人尚未终结的民事诉讼/仲裁：基于债务人财产的诉讼
      1. 首先中止；管理人接管财产掌握情况后再继续进行
      2. 个别清偿诉讼(x3+1)：中止审理 + 不予受理新的
        - 行使代位权
        - 出资不实/抽逃出资
        - 法人人格否认
        - 其他要求个别清偿的诉讼

### 第三节 管理人制度 \*选择题
- 管理人的资格：
  1. 清算组、社会中介机构、社会中介机构中的从业者；不得担任情形(x3+1)：
    + 因故意犯罪受过刑事处罚
    + 曾被吊销相关专业执业证书
    + 与本案有利害关系
    + 其他
  2. 个人担任管理人：
    + 事实清楚、债权债务关系简单、债务人财产相对集中的破产案件（没有数额小）
    + 可以聘任必要工作人员
    + 应当参加执业责任保险
  3. 利害关系回避问题(x2类)：机构/自然人
    + 机构有下列情形之一的(x4+1)：
      1. 与债务人/债权人有未了结债权债务关系
      2. 受理前3年内，为 **债务人** 提供相对固定的中介服务
      3. 现在/受理前3年内，是 **债务人/债权人** 的控股股东/实际控制人
      4. 现在/受理前3年内，担任 **债务人/债权人** 的财务/法律顾问
      5. 其他
    + 自然人情形(x4+2+1)：清算组/社会中介机构派出人员、个人管理人
      1. 有机构的4条情形
      2. 现在/受理前3年内，担任 **债务人/债权人** 的董监高
      3. 与 **债务人/债权人** 的控股股东/董监高有 夫妻/直系血亲/三代以内旁系血亲/近姻亲关系
      4. 其他
- 管理人的指定：
  1. 由人民法院指定
  2. 不能公正执行职务/不能胜任职务，债权人会议可以申请法院更换
- (\*)管理人的报酬：
  1. 由人民法院确定；是纯报酬，不包括履行职务产生的费用
  2. 根据最终清偿财产价值总额，按比例限制(上限)分段确定报酬(x7段)：物权担保人优先受偿的担保物价值 **不计入** 该总额；高院可以浮动30%制定当地限制范围+公告+最高院备案
    + (0, 100w]：≤12%
    + (100w, 500w]：≤10%
    + (500w, 1kw]：≤8%
    + (1kw, 5kw]：≤6%
    + (5kw, 10kw]：≤3%
    + (10kw, 50kw]：≤1%
    + (50kw, +∞)：≤0.5%
    + 记忆：总额1+5递增；比例12%-2公差x4 + 3/1/0.5
  3. 债务人没钱/不足以支付破产费用：根据工作时间等确定
  4. 政府部门派出工作人员参与工作：不收取报酬
  5. 管理人对担保物维护/变现/交付有合理劳动，有权向 **担保权人** 收取适当报酬；不能协商则按之前比例≤110%确定
  6. 管理人可以聘用必要工作人员，费用从破产费用列支；防止重复计酬(x2)：所需报酬从管理人报酬中支付
    + 律师/会计师事务所聘用本专业其他机构/人员
    + 破产清算事务所聘用其他机构/人员
  7. 债权人会议对管理人报酬有异议，应向法院书面提出请求和理由
- 管理人的职责与责任：
  1. 职责(x8+1)：
    + 接管财产/账簿/印章
    + 调查财产状况，制作报告
    + 决定债务人内部管理事务
    + 决定债务人日常/必要开支
    + 第一次债权人会议前，决定继续/停止营业(之后由债权人会议决定)
    + 管理/处分财产
    + 代表债务人参加法律程序
    + 提议召开债权人会议
    + 其他
  2. 对人民法院负责：人民法院的看门狗
    + 第一次债权人会议前，决定是否营业/(9+1)种重大财产处分 应经人民法院许可
    + 向法院报告工作

### 第四节 债务人财产
- 一般规定：
  1. 债务人财产的范围：宣告后改称破产财产，但范围一致；界定时点是申请受理时
    + 破产申请受理时属于债务人的全部财产
    + 破产申请受理后至破产程序终结前债务人取得的财产
    + 一些解释：
      1. 包括货币/实物/可以用货币估价和转让的财产权利
      2. 不应认定为债务人财产的(x3+1)：
        - 基于仓储/保管/借用/寄存/租赁(两种都不算)等合法占有而无所有权的他人财产
        - 在所有权保留买卖中尚未取得所有权的财产
        - 所有权专属于国家且不得转让的财产
        - 其他
      3. 设定担保物权的财产 **属于** 债务人财产；执行回转的财产属于债务人财产
      4. 共有财产中的份额：属于
        - 宣告破产清算，属于共有财产分割的法定事由
        - 因分割共有财产导致其他共有人损害产生的债务：共益债务(6#)
  2. 债务人财产的收回：出资/董监高非正常收入&侵占/质物&留置物
    + 债务人的出资人未完全履行出资义务，管理人应要求立即缴纳认缴的出资，**不受出资期限限制**：
      1. 未到期未缴/到期未缴/缴纳后抽逃
      2. 必须立即缴足认缴的出资，不论是否到期
      3. 未到缴纳期限/超过诉讼时效 的抗辩无效
      4. 法院支持 发起人/董&高/协助抽逃出资的其他股东/董&高/实际控制人承担相应责任，归入债务人财产
    + 追回董监高从企业获取的非正常收入和侵占的企业财产：
      1. 非正常收入(x2+1)：
        - 绩效奖金
        - 普遍拖欠职工工资情况下的工资性收入
        - 其他
      2. 处理方法：
        - 返还1&3项形成的债权：作为普通破产债权
        - 返还第2项形成的债权：职工平均工资
          1. 按照该企业职工平均工资计算的部分：作为拖欠职工工资清偿
          2. 超出部分：作为普通破产债权清偿
    + 通过清偿债务/提供相应担保，取回质物/留置物：
      1. 替代担保以该质物/留置物当时的市场价值为限
      2. 该财产处分应报告债权人委员会/人民法院
- **破产撤销权与无效行为**：为了公平有序清偿；由管理人统一行使
  1. 涉及债务人财产的下列行为无效(x2)：仅为突出对破产欺诈行为的打击而强调；不看时间期间
    + 为逃避债务而隐匿/转移财产的(只要求有客观后果)
    + 虚构债务/承认不真实债务的
  2. 破产撤销权：追回欺诈逃债/损害公平清偿行为的财产；民法/合同法上的撤销权不冲突时可并行适用，冲突时破产撤销权优先适用；程序判断原则
    + 人民法院受理破产申请前1年内(**唯一的标准**)，涉及债务人财产的下列行为(x5)，管理人有权请求撤销：“不正当减少”
      1. 无偿转让财产的：广义的财产；不限于转让，无偿设置用益物权等也包括
      2. 以明显不合理的价格进行交易的：30%；撤销后双方返还，产生的债务作为共益债务(6#)
      3. 对没有财产担保的债务提供财产担保的：指对已成立债务补充设置物权担保；新设带担保债务不算
      4. (\*)对未到期债务提前清偿的：“未到期”含义(x2)
        - 破产申请受理时尚未到期
        - 破产申请前到期 + 提前清偿行为发生在受理前6个月 + 债务人已发生破产原因(危机期间)
      5. 放弃债权的：放弃债权、任凭诉讼时效期间届满而不作为等
        - 债务人对外债权的诉讼时效自破产申请受理之日起中断
        - (\*)债务人无正当理由不及时行使权利，导致对外债权在破产申请受理前1年内超过诉讼时效的，破产申请受理之日起重新计算诉讼时效期间
    + 合同法上的债的保全撤销权在无冲突情况下可以并行行使：管理人未依法行使破产撤销权，债权人可以依合同法提起保全撤销权(x3情形, 1/2/5)，要求追回财产归入债务人财产(并且 **无** 本人债权范围限制)
      1. 无偿转让财产
      2. 以明显不合理价格交易
      3. 放弃债权
    + 危机期间(受理前6个月+已有破产原因) 对无物权担保的债权人个别清偿(不论何时到期/是否提前清偿)，管理人有权请求撤销；个别清偿使债务人受益(x2+1)的除外：
      1. 对有物权担保债权人在担保物市价范围内所作清偿不受此限制
      2. 清偿使债务人受益情形(x2+1)：不能撤销
        - 维系基本生产需要的水电费
        - 支付的劳动报酬/人身损害赔偿金
        - 其他受益情形
      3. 履行生效的法律文书(诉讼/仲裁/执行程序)而进行的个别清偿不能撤销；债权债务人恶意串通的有权撤销
    + 经行政清理/强制清算程序转入破产程序的，可撤销行为起算时间：
      1. 来自行政清理程序：行政监管机构作出撤销决定之日
      2. 来自强制清算程序：人民法院裁定受理强制清算申请之日
  3. 破产清算程序终结后2年内，行使破产撤销权/针对无效行为追回的财产用于对全体债权人分配；2年后对追回财产的债权人个别清偿
- (\*)取回权：权利人取回债务人占有的无所有权财产
  1. 一般取回权：破产申请受理后，债务人占有的不属于债务人的财产，权利人可以通过管理人取回(重整期间须依事先约定)；须依约交付相应费用
    + 管理人不认可取回权，权利人可以债务人为被告提起诉讼
    + 行使取回权应在破产财产变价方案/和解协议/重整计划提交债权人会议表决前提出；此后提出的应负担增加的费用
    + 债务人占有的不易保管/不及时变现价值将严重贬损的财产，及时变价并提存；对变价款行使取回权
    + 原物被违法转让：
      1. 相对人已善意取得：产生对原权利人债务
        - 转让在破产申请受理前：普通破产债权
        - 转让在受理后：共益债务(5#)
      2. 相对人已支付价款但未取得所有权：原权利人追回 + 产生对相对人返还对价的债务
        - 转让在破产申请受理前：普通破产债权
        - 转让在受理后：共益债务(5#)
    + 原物毁损灭失，获得保险金/赔偿金/代偿物：
      1. 代偿物尚未交付给债权人/已交付但能区分：取回代偿物(代偿取回权)
      2. 不能与债务人财产相区分/未取得代偿物/代偿物价值不足弥补：产生对原权利人债务
        - 毁损灭失在破产申请受理前：普通破产债权
        - 毁损灭失在受理后：共益债务(5#)
      3. 相关责任人赔偿：
        - 管理人/相关人员执行职务时，故意重大过失 不当转让他人财产/造成他人财产毁损灭失：
          1. 共益债务的补充赔偿责任
          2. 债务人财产减少的赔偿责任
        - 董&高：受理前非法转让他人财产，相应赔偿责任
  2. 出卖人取回权：
    + 未付清全部价款 + 已发运未收到：出卖人可以取回在途标的物；方式(x2)
      1. 通知承运人/实际占有人：中止运输/返还货物/变更目的地/转交他人
      2. 在货物未达管理人前已向管理人主张取回，到达后可取回
    + 未及时行使取回权，到达后再向管理人主张取回的，不应允许
  3. **所有权保留买卖合同的处理**：所有权未转移前一方破产，属于双方均未履行完毕的合同；哪方破产/继续or解除
    + 出卖人破产 + 继续履行：买受人继续履行
      1. 买受人未依约履行/不当处分标的物(此时买受人未取得所有权)：出卖人可以主张取回标的物
      2. 买受人已支付总价款≥75%/第三人善意取得标的物：不可取回，可主张买受人继续履行+赔偿责任
    + 出卖人破产 + 解除合同：
      1. 可要求返还标的物，不支持买受人抗辩；产生对买受人已支付价款债务(好人/坏人)
      2. 买受人合同履行中依法履行者：共益债务(1#)
      3. 违反约定义务：普通破产债权
    + 买受人破产 + 继续履行：
      1. 买受人履行义务期限(支付价款等)视为到期，应及时履行
      2. 无正当理由不履行/不当处分：出卖人可主张取回标的物
      3. 买受人已支付总价款≥75%/第三人善意取得标的物：不可取回，可主张买受人继续履行+赔偿责任
      4. 上述继续履行+赔偿责任：共益债务(1#)
    + 买受人破产 + 解除合同：
      1. 双方返还
      2. 取回标的物价值明显减少，出卖人可从已支付价款中优先抵扣
      3. 不足抵扣，形成债务：共益债务(1#)
- 抵销权：**管理人/债务人不得主动主张** 债务抵销(相当于个别清偿有失公平，受益除外)；受理前即互负债权债务，无论是否到期、标的种类品质是否相同，债权人均可主张抵销
  1. 债权人应在破产财产最终分配确定前主张
  2. 管理人对抵销主张有异议的，在约定异议期/收到之日起3个月内诉讼；下列情形(x3)不予支持：
    + 破产申请受理时任一债务未到期
    + 互负债务标的物种类品质不同
    + 无正当理由逾期提出异议
  3. 判决驳回抵销无效诉讼请求/管理人无异议：抵销自收到主张通知之日起生效
  4. 不得抵销情形(x3+2)：抵销权是一种不太公平的个别清偿，故需限制故意取得(受理后/已知/申请1年前除外)+股东x2；别除权人担保物市场价值内抵消无担保债务不受此限
    + 债务人的债务人在破产申请受理后取得他人对债务人的债权的
    + 债务人的债务人已知债务人有破产原因/申请破产，对债务人取得债权的；因法律规定/破产申请1年前原因的除外
      1. 一般是通过受让他人债权(才存在取得成本与抵消清偿的差额)
      2. 也有通过不合理价格交易等方法(20w旧设备100w卖给债务人等)
    + 债权人已知债务人有破产原因/申请破产，对债务人负担债务的；法律规定/申请1年前原因除外
    + 债务人股东主张以其债权抵销 因欠缴/抽逃出资对债务人所负债务
    + 债务人股东主张以其债权抵销 因滥用股东权利/关联关系损害公司利益对债务人所负债务
  5. 危机期间违规抵销，管理人可以申请认定抵销无效：危机期间债务人与个别债权人抵销，且其抵销的债权债务属于上述不得抵销情形2&3的，管理人应在受理后3个月内向法院主张抵销无效
- 破产费用与共益债务：
  1. 破产费用：受理申请后，破产程序进行发生的费用
    + 破产案件诉讼费用
    + 管理/变价/分配债务人财产的费用
    + 管理人/工作人员履职费用和报酬
  2. 共益债务(x6)：破产申请受理后发生的
    + 管理人/债务人请求继续履行双方均未履行完毕的合同所产生的债务
      1. 所有权保留买卖，出卖人破产+解除合同：买受人依约履行义务的(好人)
      2. 所有权保留买卖，买受人破产+选继续履行+但无正当理由不履行/不当处分：继续履行(支付价款)+赔偿责任
      3. 所有权保留买卖，买受人破产+解除合同+双方返还：标的物价值明显减少，已支付价款不足抵扣形成的债务
    + 债务人财产受无因管理产生的债务
    + 债务人不当得利产生的债务
    + 债务人继续营业而应支付的劳动报酬/社会保险费用/由此产生的其他债务(新产生的)
      1. 受托人已知破产，为债权人利益在无法移交事务的紧急情况下继续处理委托事务产生的债权
    + 管理人等执行职务致人损害产生的债务：
      1. 合法占有而无所有权之物，在破产申请受理 **后** 被违法转让(善意取得/已支付价款)
      2. 合法占有而无所有权之物，在破产申请受理 **后** 毁损灭失且能取回的代偿物不足(混同/没有/不足)
    + 债务人财产致人损害产生的债务：
      1. 分割共有财产导致其他共有人损害的
      2. 行使破产撤销权，撤销明显不合理价格交易产生的返还债务
  3. 破产费用(2.1)与共益债务(2.2)的清偿：
    + 破产费用和共益债务由债务人财产 **随时清偿**
    + 不足以清偿所有破产费用 **和** 共益债务的：先清偿破产费用(两个都有，都还不上，先还破产费用)
    + 不足以清偿所有破产费用 **或** 共益债务的：按照比例清偿(只存在一个还不上，每个债权人按比例)
    + 债务人财产不足以清偿破产费用的：
      1. 管理人提请终结破产程序
      2. 人民法院收到请求15日内裁定终结
      3. 有人愿意垫付的，可以继续程序
      4. 此类没钱案件应优先受理

### 第五节 破产债权 \*重要
- 破产债权申报的一般规则：
  1. 债权申报期限：申请受理公告之日起算，[30日, 3个月]，具体由人民法院确定
  2. **债权申报的要求**：在规定期限内申报；只有职工债权不要申报
    + 债务人拖欠职工工资/应划入个人账户的社保金：**不必申报**
      1. 工资
      2. 医疗伤残补助
      3. 抚恤费用
      4. 基本养老保险
      5. 基本医疗保险
      6. 补偿金
    + 税收/社会保障/别除权人的债权都要申报
    + 未到期债权，在破产申请受理时视为到期
    + 付利息债权(贷款等)自破产申请受理时停止计息
    + 无利息债权无论是否到期均以本金申报
    + 附条件/期限的债权和诉讼仲裁未决的债权也可申报
    + 连带债权人可由一人代表，也可共同申报
    + 超过期限的，可在最后分配前补充申报；之前分配完的不补充分配
    + 连带债务人数人的破产案件均被受理的，债权人有权就 **全部债权分别** 在各破产案件中申报；接受清偿总额不得超过其债权总额
    + 解除双方均未履行完毕的合同所产生的损害赔偿请求权，申报普通债权；以实际损失为限，违约金不得申报(一笔勾销)
    + 债务人是委托合同的委托人，破产案件被受理：
      1. 受托人不知该事实，继续处理委托事务产生的债权：普通破产债权
      2. 受托人已知该事实：
        - 为了债务人/全体债权人利益，在无法移交的紧急情况下继续处理委托事务产生的债权：共益债务(4#)
        - 无必要地继续处理委托事务，不当增加的费用/报酬：**不得受偿**
    + 债务人是出票人，破产申请被受理后付款人继续付款/承兑的：普通破产债权
- 破产债权申报中的保证：
  1. 被保证人(债务人)破产：债权人以余额要求承担保证责任的，最迟应在破产程序终结后6个月内提出
    + 连带责任保证人：
      1. 可以直接要求保证人清偿/先要求债务人清偿再以余额向保证人追偿/同时向债务人和保证人追偿
      2. 债务人先清偿的，相应减少保证人的保证责任；保证人先清偿的，相应破产债权转移给保证人行使
      3. 债权人未申报债权时允许连带债务人/保证人预先申报债权；债权人已申报的保证人不能再申报
      4. 债权人(应当)知道债务人破产，既不申报债权也不通知保证人，使保证人不能预先行使追偿权的，保证人在该债权在破产程序中可能受偿的范围内免除保证责任
    + 补充责任保证人(一般保证人)：先诉抗辩权(x)/期限利益(√)
      1. 债务人破产案件被受理，不得行使先诉抗辩权；债权人可直接向一般保证人追偿
      2. 仍享有期限利益：受理时主债务未到期，无提前履行义务(仍按照原约定期限承担保证责任)
  2. 保证人破产：保证责任不因破产而免除；期限利益(x)
    + 连带保证人：未到期的债务也视为已到期，直接申报债权求偿即可
    + 一般保证人：先诉抗辩权(x)/补充责任(√)
      1. 先以全部保证范围申报债权，获得清偿后先 **提存**
      2. 待债务到期从债务人处受偿后，再以实际应承担补充责任的破产债权数额支付，余款由法院收回分配给其他债权人
      3. **举例**：
        - 债权10w，全额一般保证
        - 保证人先破产，申报时推定全额，分配比例50%，获得5w提存
        - 债务到期后债务人也破产，分配比例50%，获得5w
        - 此时一般保证人的保证责任只有5w，分配比例50%，**实际清偿额应为2.5w**，剩余法院收回继续分配
        - 共得到7.5w清偿；可以看出这样的结果和债务人先破产保证人后破产是一样的，这也就是立法目的
- 破产债权确认：
  1. 管理人审查确认
  2. 债权登记表提交第一次债权人会议核查
  3. 债权确认表由人民法院裁定确认

### 第六节 债权人会议
- 概念/成员/权利：破产议事机构
  1. 申报债权者均有权参加第一次债权人会议，后续只有债权得到确认者才有表决权
  2. 有物权担保未放弃的债权人属于债权人会议成员，但对通过和解协议&通过破产财产分配方案无表决权(和TA没关系的事项)
  3. 应有职工/工会代表参加债权人会议发表意见，但对不影响其清偿利益的事项无表决权
  4. 债权人会议设主席一人，由人民法院在有完全表决权的债权人中指定(通常是无优先权的大债权人)
- 召集与职权：
  1. 召集：
    + 第一次由人民法院召集，债权申报期限届满15日内召开
    + 后续(x4类人)向会议主席提议召开：有提议即召开，无权拒绝；管理人提前15日通知
      1. 人民法院
      2. 管理人
      3. 债权人委员会
      4. 占(所有的)债权总额≥1/4以上债权人
  2. 职权(x10+1)：
    + 核查债权
    + 申请更换管理人/审查费用报酬
    + 监督管理人
    + 选任/更换债权人委员会
    + 继续/停止营业
    + 通过重整计划
    + 通过和解协议
    + 通过财产管理方案
    + 通过破产财产变价方案
    + 通过破产财产分配方案
    + 其他
  3. 决议方式：对在该事项有表决权的全体债权人有约束力
    + 一般决议(>1/2人+≥1/2钱)：包括通过破产财产分配方案
      1. 出席会议有表决权债权人过半数通过
      2. 且代表无财产担保债权总额1/2以上
    + 通过和解协议 & 重整计划：>1/2人+≥2/3钱
      1. 和解协议(>1/2人+≥2/3钱)：
        - 出席会议有表决权债权人过半数通过(此处物权担保人无表决权&不受约束)
        - 且代表无财产担保债权总额2/3以上
      2. 重整计划：分组表决，每组>1/2人+≥2/3钱+人民法院批准
    + 对决议有异议：作出后15日内请求裁定撤销
  4. 债权人会议僵局解决：
    + 僵局定义(x2)：发生后由人民法院裁定
      1. 债务人财产管理方案/破产财产变价方案 一次表决未通过
      2. 破产财产分配方案 二次表决未通过
    + 对裁定不服的(x2)：宣布/收到通知15日内申请复议(不能上诉)
      1. 对1#情况裁定不服的：任何债权人
      2. 对2#情况裁定不服的：债权额占无财产担保债权总额≥1/2以上的
- 债权人委员会：破产监督人
  1. 选任机关：≤9人的奇数，有一名职工代表；未设的由人民法院代劳
  2. 职权(x3+1)：
    + 监督债务人财产管理/处分
    + 监督破产财产分配
    + 提议召开债权人会议
    + 其他
  3. 管理人实施下列行为(x9+1项重大财产处分)，应及时报告债权人委员会：
    + 不动产转让
    + 财产权转让
    + 全部库存/营业转让
    + 借款
    + 设定财产担保
    + 债权/有价证券转让
    + 决定履行双方均未履行完毕的合同
    + 放弃权利
    + 担保物收回
    + 其他

### 第七节 重整程序
- 挽救一段时间；申请时间可提前到有破产原因发生的可能时即可提出
- 重整申请：
  1. 债权人/债务人直接申请重整
  2. 债权人申请破产的，受理后宣告前：债务人/出资占注册资本10%以上的出资人/其他债权人 可以申请重整
  3. 债务人申请破产自己想死，别人不能申请重整
- 重整期间：
  1. 仅指重整申请受理 至 重整计划草案分组表决通过+人民法院审查批准/不通过/不批准 的期间；**不包括** 重整计划的执行期间
  2. 财产管理/事务执行，可由债务人/管理人负责
  3. 债权人的财产担保权暂停行使；无保留必要的担保财产可行使担保权；为继续营业而借款的可以新设担保
  4. 为重整发生的债务/费用，属于共益债务，随时受偿；取回权行使须符合约定条件
  5. 不得分配股利；董监高不得对外转让持有的债务人股权(人民法院同意除外)
  6. 有下列情形(x3)之一的，经请求应裁定终止重整+宣告破产：
    1. 情况继续恶化，缺乏挽救可能性
    2. 债务人有欺诈/恶意减少财产/其他显著不利于债权人的行为
    3. 由于债务人行为使管理人无法执行职务
- 重整计划的制定：谁管事谁制定
  1. 裁定重整6个月内，正当理由可延期3个月
  2. 逾期不提出，裁定终止重整+宣告破产
  3. 控股股东/实际控制人/其关联人对上市公司造成损害的，重整计划草案应根据其过错将其股权相应减少调整
- 重整计划草案的表决和批准：分组表决+法院批准；收到重整计划草案30日内召开债权人会议表决
  1. 指导性分组(x4+1)：
    + 有物权担保的债权人
    + 职工债权人
    + 税务机构
    + 普通债权人
    + 出资人组：有出资人权益调整事项时；该组表决按股东(大)会特别决议规则(参与表决+≥2/3)
  2. 债权人组表决方式：>1/2人+≥2/3钱；出资人组按出资比例，不看人数
    + 同一表决组债权人过半数同意
    + 且代表该组债权总额≥2/3以上
  3. 重整计划不得规定减免债务人欠缴的纳入社会统筹账户的社保费用，该项债权人不参加草案表决
  4. 人民法院批准草案：全部通过批准/强制批准
    + 均须审查反对者(反对组/通过组的反对者)法定既得利益不受损害(与破产清算比较)
    + 上市公司重整涉及行政许可(借壳上市+发行股份购买资产)的，需通过最高院启动与证监会的会商机制
    + 强制批准程序：**部分未通过时**，全部未通过时人民法院闭嘴
      1. 债务人/管理人与未通过组协商；协商后拒绝再表决/再表决未通过
      2. 但草案符合下列条件(x6)，可以申请人民法院强制批准：
        - [别除权人利益] 有物权担保的债权就担保物将获得全额清偿，因延期所受损失得到公平补偿 or 该组已通过
        - [职工债权+税款] 职工债权和税款将获得全额清偿 or 相应组已通过
        - [普通债权人] 普通债权所获清偿比例不低于(≥)依破产清算程序所能获得的比例 or 该组已通过
        - [出资人] 对出资人权益调整公平公正 or 该组已通过
        - 公平对待同一表决组成员，且清偿顺序不违反法定破产清偿顺序
        - 债务人经营方案有可行性
    + 全部通过后10日内提出批准申请；正常批准/强制批准均为收到申请30日内裁定批准+终止重整程序+公告
  5. 表决与批准情形综述(x5)：未批准/全部未通过，终止+宣告债务人破产；批准的，终止+执行

|债权人会议|人民法院|重整程序|结局|
|---|---|---|---|
|全部通过|批准|终止|执行|
|全部通过|未批准|终止|宣告破产|
|部分通过|强制批准|终止|执行|
|部分通过|未批准|终止|宣告破产|
|全部未通过|闭嘴(无权发言)|终止|宣告破产|

- 重整计划的执行：
  1. 债务人执行，管理人监督；涉及上市公司须证监会行政许可的(借壳上市/发行股份购买资产)应按规定获得许可
  2. 对债务人和全体债权人(包括物权担保债权人)均有约束力；但债权人对债务人的保证人/其他连带债务人的权利不受影响
  3. (\*)未依法申报债权的债权人可以继续申报，但在重整计划执行期间不得行使权利，执行完毕后可以按照同类债权清偿条件行使权利
  4. 债务人不能执行/不执行重整计划的，经利害关系人请求应裁定终止执行+宣告破产
  5. 终止重整计划执行的效力(x4)：债权人不吃亏+不能继续优于同顺位
    + 债权人在重整计划中作出的 债权调整承诺失去效力
    + 为重整计划的执行提供的担保继续有效
    + 因执行重整计划所受的清偿仍然有效，未受清偿部分作为破产债权
    + (\*)已接受清偿的债权人，只有在同顺位债权人所受清偿达同一比例时，才能继续接受破产分配
  6. 按照重整计划减免的债务，自重整计划执行完毕时起，债务人不再承担清偿责任(一笔勾销)

### 第八节 和解制度(债务打折)
- 一般程序：
  1. 只能由债务人一方提出：直接提出/破产申请受理后宣告前提出
  2. 通过和解协议的决议：出席会议有表决权的债权人过半数同意(>1/2人)，且其代表无财产担保债权总额≥2/3以上
  3. 有物权担保的债权人对此事项无表决权，也不受和解协议的约束
  4. 表决与批准情形：
    + 债权人会议通过 + 人民法院认可：终止和解程序 + 公告
    + 债权人会议通过 + 人民法院不认可：终止和解程序 + 宣告破产
    + 债权人会议未通过 + 人民法院闭嘴：终止和解程序 + 宣告破产
- 和解协议的效力：
  1. 对债务人和全体和解债权人均有约束力：指无物权担保债权人；别除权人不受约束
  2. 按照和解协议减免的债务，和解协议执行完毕时起，债务人不再承担清偿责任
  3. 未在期限内申报债权的可以继续申报，但在和解协议执行期间不得行使权利；执行完毕后可以按照协议规定的条件行使权利
  4. 和解协议对债务人的保证人/连带债务人无效
- 和解协议的终止：
  1. 债务人不能执行/不执行和解协议，经和解债权人请求应裁定终止执行 + 宣告破产
  2. 终止和解协议执行的效力(x4)：同重整
    + 债权调整承诺失去效力
    + 为和解协议执行提供的担保继续有效
    + 因执行所受清偿有效，未受清偿部分作为破产债权
    + 已受清偿的债权人，只有在同顺位债权人所受清偿达同一比例时，才能继续接受破产分配

### 第九节 破产清算程序
- 别除权：物权担保，不包括保证担保
  1. 对破产人特定财产(物权)享有担保权的权利人，对该特定财产享有优先受偿的权利
  2. 属于破产债权，担保物属于破产财产，需要申报债权；清偿顺序(**就该特定财产**)最优先
  3. 债务人以自己财产设定担保，别除权人行使优先受偿权利，未受偿部分作为普通债权；放弃优先权则变成普通债权
  4. 破产人仅为担保人为他人债务提供物权担保：此时两人之间只有担保关系，而无基础债权债务关系
    + 担保债权人的债权构成别除权；担保物价款不能足额清偿时，余债只能向主债务人求偿
    + 别除权人若放弃优先权，也不能转为对担保人的普通债权
    + 总的来说物权担保人破产时就担保财产仁至义尽即可
  5. 特别优先权：**优于抵押权和其他债权**
    + 建设工程款债权就该工程折价/拍卖的价款优先受偿
    + 划拨所得土地使用权拍卖后优先缴纳土地使用权出让金
- 破产财产的变价和分配：
  1. 破产财产清偿顺序：不足清偿同一顺序的，按比例分配
    + 1. 有财产担保的债权
    + 2.1 破产费用 2.2 共益债务
    + 3. 职工债权：董监高工资按企业职工平均工资计算
    + 4. 其他社保费用(社会统筹账户)和欠缴税款
    + 5. 普通破产债权
  2. 欠缴税款产生的滞纳金：
    + 受理前产生的：普通破产债权
    + 受理后产生的：不属于破产债权，不予清偿
  3. 商业银行破产清算顺序特别规定：清算费用、职工工资、劳动保险费用后，优先支付个人储蓄存款的本金和利息
  4. 农民专业合作社破产清算顺序特别规定：破产费用+共益债务后，优先清偿破产前与农民成员已发生交易但尚未结清的款项
  5. 管理人拟定破产财产分配方案，债权人会议表决通过(一般决议)，人民法院裁定认可后管理人执行
  6. 交付困难处理：
    + 无法直接交付且通知不到/通知没反应的：提存 + 最后分配公告日2个月后仍不领取视为放弃
    + 附生效/解除条件的债权，提存 + (仅)在最后分配公告日查看条件是否成就，以此决定分配给该债权人 or 其他债权人
    + 诉讼/仲裁未决的债权：提存 + 破产程序终结之日满2年仍不能受领的，分配给其他债权人
- 破产程序的终结：
  1. 终结方式(x4)：
    + 和解/重整顺利完成
    + 清算外方式解决清偿问题(第三人代为清偿/自行和解)
    + 债务人破产财产不足以支付破产费用
    + 破产财产分配完毕
  2. 人民法院自收到管理人终结请求15日内裁定，裁定后10日内注销登记
  3. 终结之日起2年内有下列情形(x2)的，可以请求按照分配方案追加分配：不足支付分配费用的直接上交国库；2年后对追回财产的债权人个别清偿
    + 发现可撤销/无效行为/债务人董监高非正常收入/侵占财产，应当追回的
    + 发现破产人有应当供分配的其他财产的

## 第九章 票据与支付结算法律制度
小题不多，综合题必有一道；本章按轻一整理

- 票据无效的情况：
  1. 记载了支付条件的(欠缺绝对必要记载事项)
  2. [“金收日”不得更改] 票据金额、收款人名称、出票日期更改的
  3. 票据金额中文大写和数码不一致的
  4. 出票人签章不符合规定的
- 票据分类和提示付款/承兑：
  1. 支票：银行/金融机构为付款人 → 并非票据债务人；仅限见票即付
  2. 本票(承诺自己见票即付的)：仅限银行本票，出票人本人付款；仅限见票即付；未按照规定期限提示付款，丧失对出票人以外的前手的追索权
  3. 汇票：
    + 银行汇票：仅限见票即付
    + 商业汇票：个人(x)，只有这里才有远期和承兑的问题
- 背书：
  1. 附条件的背书：背书有效，条件没有票据法上的效力
  2. 部分背书/多头背书：背书无效，票据权利未转移
  3. **“不得转让” 记载的效力**：出票人/背书人
    + 出票人记载的：转让无效，票据权利未转移
    + 背书人记载的：转让有效，但记载人仅对直接后手承担保证责任；记载人的正常前手仍应承担责任
  4. **不得背书转让**[带现金的x3]：填明“现金”字样的银行汇票、银行本票和现金支票
- 承兑：
  1. 收到提示承兑的汇票3日内无表示，视为拒绝承兑
  2. 承兑附条件：视为拒绝承兑
  3. 承兑人为主债务人，不得以其与出票人之间的资金关系对抗持票人，拒绝支付票据金额
- 保证：
  1. 票据保证人与被保证人负有同一责任(A=B)，无先诉抗辩权
  2. 保证人2人以上，为连带责任
  3. 附条件的保证，保证有效，条件无票据法效力
  4. 保证人清偿债务后，可以对被保证人及其前手行使追索权
  5. 被保证人的债务因形式要件欠缺而无效(票据无效，没有票据债务了)，保证人也不承担票据责任
  6. 被保证人的债务因实质要件欠缺而无效(票据有效，票据债务还有，只是被保证人脱身了)，保证人仍应承担票据责任
- 付款：
  1. 审查背书连续性
  2. 错误付款：
    + 善意无重大过失：全部票据关系消灭
    + 恶意/重大过失：票据关系没有消灭
- 追索权：取得相关证明(破产宣告文书/退票通知书等) → 行使追索权
  1. 到期追索权
  2. 期前追索权：仅限承兑人的问题
  3. 没有相关证明的，只能向出票人和承兑人追索，对其他所有的前手(背书人&保证人)都没有追索权了
  4. 消灭时效：对其他前手
    + 首次追索权：6个月
    + 再追索权：3个月
  5. 追索金额：汇票金额、利息、费用；不包括间接损失
  6. 追索对象：出票人、承兑人、背书人、保证人责任连带；想找谁找谁
    + 可不按顺序向其中一人、数人或者全体行使
    + 已经开始追索的，仍然可以继续向其他人追索
  7. 回头背书的追索权：
    + 持票人为出票人的(大回头)，对前手都没追索权了
    + 持票人为背书人的，对后手都没追索权了
- 票据权利的取得：
  1. 善意+对价持票人，100%票据权利：不受其前手权利瑕疵及前手间相互抗辩的影响
  2. 无对价/无相当对价取得票据的：若属善意取得，其权利不得优于其前手
- 代行：签别人的名字盖别人的章
  1. 有权：有权代理
  2. 无权：伪造
  3. 表见代行：类推表见代理，本人承担责任
- 票据行为的代理：明示以被代理人名义，签代理人的章；有效代理下被代理人承担责任
  1. 越权代理：超越部分代理人担责，授权部分被代理人担责
  2. 狭义无权代理：
    + 追认了就同有权
    + 没追认的话，**票据权利未产生**，所以大家都没票据责任
    + 但如果持票人转让，对方善意取得，则无权代理人要担责(他自己签了章的)
  3. 表见代理：还是一样
- 票据伪造变造：
  1. 伪造：签章是假的 假冒/虚构
    + 伪造人自己没签章，所以没有票据责任
    + 被伪造人无辜，也不承担票据责任
    + 不影响真实签章效力
    + 明知伪造的，不能获得票据权利
  2. 变造：无权更改的人偷偷摸摸更改签章以外的记载事项
    + 变造前签章的对原记载事项负责
    + 变造后签章的对变造后记载事项负责
    + 分不清什么时候签章的，视同在 **变造前** 签章
- 票据原因关系：真实的交易关系和债权债务关系；贴现除外
  1. 没有基础关系：出票和背书无效，行为接收方不取得票据权利
  2. 但是有形式上的权利，与善意取得结合
  3. 票据权利买卖不属于真实交易关系，贴现除外
  4. 例外：
    + 税收/继承/赠与等依法无偿取得，持票人依法举证即可；权利不优于直接前手
    + 向金融机构贴现可以
- **票据抗辩**：
  1. 对物抗辩：绝对抗辩(可以对任何持票人抗辩)
    + 全部票据权利均不存在
    + 票据上记载的特定债务人的债务不存在：我并非票据债务人
      1. 签章人民事行为能力不完全
      2. 狭义无权代理的被代理人
      3. 被伪造人
      4. 变造前的签章人对变造后的内容
      5. 票据权利时效期间届满，票据债务消灭
      6. 未进行票据权利保全(没有退票通知书等证据)
  2. 对人抗辩：相对的抗辩(对特定持票人抗辩)
    + 对不履行约定义务的 与自己有直接债权债务关系的持票人抗辩
    + **不得** 以与出票人之间的抗辩事由对抗持票人
    + **不得** 以与持票人的前手抗辩事由对抗持票人，除非持票人明知(交付时)
    + 善意取得者权利100%
    + 无对价/无相当对价，权利不得优于前手
  3. 善意+完全对价：有抗辩切断的效果
- 票据权利补救：挂失止付/公司催告/普通诉讼
  1. 挂失止付：
    + 可以挂失止付的票据：有付款人的
      1. 支票
      2. 填明现金字样的银行本票
      3. 填明现金字样和代理付款人的银行汇票
      4. 已承兑的商业汇票
    + 地位：非必经程序，3日内申请公示催告/普通诉讼(可以不经挂失止付直接催告/诉讼)；挂失止付通知12日内有效
  2. 公示催告：申请 → 一定期限 → 除权判决
    + 不得背书转让的3种带现金的票据不能公示催告
    + 受理后3日内发公告 → 公示期间不得少于60日 → 催告期间届满无裁定终结事由 → 1个月内申请除权判决
    + 正当理由未及时申报，(应当)知道判决公告之日起1年内起诉撤销除权判决
- 非票据结算方式：
  1. 汇兑：企业(√) 个人(√)
  2. 托收承付：企业(√) 个人(x)
    + 异地
    + 商品交易及其附属劳务
    + 验单付款发出后3日；验货付款发出后10日
    + 承付期经过，默示同意付款(√)
  3. 委托收款：接到后3日，默示同意付款(√)；企业(√) 个人(√)
  4. **国内信用证**：企业(√) 个人(x)
    + 不可撤销、不可转让的跟单信用证
    + 不得支取现金
    + 信用证只处理单据；受理时收取不低于20%保证金
    + **开证行是主债务人**，申请人账户不足部分作逾期贷款处理

  1. 核准类账户
  2. 备案类账户
