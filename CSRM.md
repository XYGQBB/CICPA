## 第一章 战略与战略管理
细节选择题为主；厕所文学

### 第一节 公司战略的基本概念
- 公司战略的定义
  1. 传统概念<波特>
    + 终点+途径
    + 计划性、全局性、长期性
  2. 现代概念
    + 计划性+非计划性<明茨伯格>
    + 仅途径
    + 预先性(预谋战略)+反应性(适应性战略)<汤姆森>
    + 应变性、竞争性、风险性
- 公司使命
  1. 公司目的：营利/非营利
  2. 公司宗旨：经营范围&定位；用词无具体数据/期限/指标，大&空
  3. 经营哲学：价值观、信念、行为准则
- 公司目标
  1. 使命具体化，注意与使命区别
  2. 有具体的完成期限/定量化/具体指标
  3. 战略目标 & 财务目标
- 公司战略的功能(x3)
  1. 指明了企业的发展方向
  2. 是整合和优化企业资源能力的依据和动力
  3. 是提升企业管理效能的前提和保障
- 公司战略的层次
  1. 总体(公司层)战略
  2. 业务单位(竞争)战略：业务多元化公司才有分此层的意义
  3. 职能战略

### 第二节 公司战略管理
- 战略管理过程
  1. 战略分析
    + 外部环境分析：宏观、产业、竞争
    + 内部环境分析：企业资源、能力、核心竞争力；波士顿矩阵/通用矩阵/SWOT分析
  2. 战略选择
    + 可选类型：
      1. 总体战略：发展、稳定、收缩
      2. 业务单位战略：成本领先、差异化、集中化
      3. 职能战略(x6)：市场营销、研发、生产运营、采购、人力资源、财务管理
    + 选择过程(x4)：
      1. 制定战略选择方案：集权分权程度的把握
        - 自上而下的方法
        - 自下而上的方法
        - 上下结合的方法
      2. 评估战略备选方案
        - 适宜性标准
        - 可接受性标准
        - 可行性标准
      3. 选择战略
        - 根据企业目标
        - 提交上级管理部门审批
        - 聘请外部机构
      4. 战略政策和计划
  3. 战略实施：
    + 有效的组织结构
    + 人员和制度管理
    + 公司政治
    + 组织协调和控制系统
    + 协调企业战略、结构、文化、控制
- **战略变革管理**：
  1. 性质：
    + 渐进性变革：常常发生/稳定变化/影响企业一部分
    + 革命性变革：不常发生/全面转化/影响整体
  2. 战略变革的发展阶段(x4)：
    + 连续阶段：基本没有大变化
    + 渐进阶段：缓慢变化，零敲碎打or系统性
    + 不断改变阶段：无方向/无重心
    + 全面阶段：较短时间内发生革命性变化
  3. 战略变革的动因：
    + 外部环境变化
    + 技术/工作方法变化
    + 产品/服务变化
    + 管理/工作关系变化
    + 组织结构/规模变化
    + 并购后
  4. 战略变革的类型：
    + 技术变革
    + 产品和服务变革
    + (组织)结构和(管理)体系变革
    + 人员变革
  5. 战略变革的时机选择：
    + 提前性变革
    + 反应性变革
    + 危机性变革
  6. 战略变革的模式：[2x2矩阵]
    + 横坐标：变革的性质 渐变性/革命性
    + 纵坐标：管理层的作用 主动/被动
    + 模式(x4)：
      1. 主动+渐变：协调
      2. 主动+革命：计划
      3. 被动+渐变：接受
      4. 被动+革命：迫使
  7. 战略变革的主要任务(xxx)：
    + 调整企业理念：思想先行
    + 重新定位企业战略
    + 重新设计企业组织结构
  8. **战略变革的实现**：
    + 推进变革的步骤(x4)：
      1. 高级管理层决定应该做什么
      2. 指定一个代理人掌握变革
      3. 变革代理人赢得关键部门管理人员的支持
      4. 变革代理人督促各管理人员立即行动并提供支持
    + 变革受到抵制的原因(x3)：
      1. 生理变化：工作模式/工作地点
      2. 环境变化：物质环境/人际环境
      3. 心理变化：
        - 迷失方向
        - 不确定性导致无安全感
        - 无力
    + 变革实现的障碍(x2)：
      1. 文化障碍：企业文化的惯性
      2. 私人障碍：
        - 习惯
        - 对个人收入的影响
        - 对未知的恐惧
        - 选择性信息处理/接受
    + 克服变革阻力的策略(x3)：答题时注意按照课本倾向
      1. 变革的节奏：鼓励循序渐进，反对激进
      2. 变革的管理方式：鼓励民主，反对集权
        - 鼓励冲突领域的对话
        - 提供培训
        - 鼓励个人参与
      3. 变革的范围：鼓励同等条件较小范围

## 第二章 战略分析
外部 + 内部环境分析；SWOT分析

### 第一节 企业外部环境分析
- 宏观环境分析：PEST模型
  1. 政治和法律因素(P)
    + 政治环境分析要素(x4)：
      1. 企业所在国家和地区的政局稳定状况
      2. 政府行为对企业的影响
      3. 执政党所持的态度和推行的基本政策，以及这些政策的连续性和稳定性
      4. 各政治利益集团对企业活动产生的影响：立法/舆论
    + 法律环境分析要素(x4)：
      1. 法律规范，特别是和企业经营密切相关的经济法律法规
      2. 国家司法机关和执法机关
      3. 企业的法律意识
      4. 国际法所规定的国际法律环境和目标国国内法律环境
    + 政治法律环境对企业战略影响的特点(x3)：
      1. 不可测性
      2. 直接性
      3. 不可逆转性
  2. 经济环境分析要素(x6, E)：
    + 社会经济结构：产业/分配/交换/消费/技术结构
    + 经济发展水平：GDP、人均、增速
    + 经济体制
    + 宏观经济政策：货币/财政/收入分配/对外经济政策
    + 当前经济状况：税收水平、通胀率、利率等
    + 其他一般经济条件：工资水平、上游企业/竞争对手价格变化等，**此项把中观微观经济条件均包括了**
  3. 社会和文化环境分析要素(x6, S)
    + 人口因素：出生率、死亡率、平均寿命等
    + 社会流动性
    + 消费心理：新鲜感、实用性
    + 生活方式变化
    + 文化传统：社会习惯(e.g. 春节/圣诞节；宗教)
    + 价值观：个人主义/集体主义
  4. 技术环境分析要素(x5, T)
    + 基本技术的进步使企业能对市场及客户进行更有效的分析
    + 新技术的出现使社会和新兴行业对本行业产品和服务的需求增加，从而使企业可以扩大经营范围或开辟新的市场
    + 技术进步可创造竞争优势
    + 技术进步可导致现有产品被淘汰，或大大缩短产品生命周期
    + 新技术的发展使企业可更多关注环境保护、企业的社会责任及可持续成长等问题
- 产业环境分析：产品生命周期、五力模型
  1. 产品生命周期(x4)：*背诵4.战略目标和路径*
    + 导入期
      1. 产品技术特点：尚在不断发展变化，质量有待提高
      2. 用户很少，营销成本高、生产成本高、净利润较低
      3. 竞争对手少，经营风险非常高
      4. 战略目标：扩大市场份额；战略路径：产品研发和技术改进
    + 成长期
      1. 产品技术特点：产品差异较大
      2. 销量增长快(成长期标志)，广告费用较高但单位销售收入分担的广告费用在下降，净利润最高
      3. 市场扩大竞争者涌入，经营风险有所下降但仍然较高
      4. 战略目标：争取最大市场份额；战略路径：市场营销
    + 成熟期
      1. 产品技术特点：逐步标准化
      2. 市场基本饱和，价格开始下降，竞争者之间出现挑衅性的价格竞争(成熟期标志)，毛利率和净利润率均下降
      3. 价格竞争激烈，经营风险继续降低到中等水平
      4. 战略目标：巩固市场份额；战略路径：提高效率，降低成本
    + 衰退期
      1. 产品差异很小
      2. 产能严重过剩
      3. 产品价格、毛利都很低，经营风险进一步降低
      4. 战略目标：防御以获取最后的现金流；战略途径：控制成本
    + 理论局限性：
      1. 每阶段持续时间随产业不同而不同，落点难以界定
      2. 产业增长并不总是呈S形
      3. 可以通过产品创新和重新定位来影响曲线形状
      4. 每阶段竞争属性随产业不同而不同
  2. **五力模型**：
    + 五种竞争力分析
      1. 潜在进入者的进入威胁
        - 影响：瓜分市场份额；减少市场集中
        - 进入障碍：
          1. 结构性障碍(客观不变的)：
            + 规模经济
            + 现有企业对关键资源的控制
            + 现有企业的市场优势：品牌优势、政府政策
          2. 行为性障碍(主观能动的反应)：报复手段
            + 限制进入定价
            + 进入对方领域
      2. 替代品的替代威胁
        - 产品替代的概念：直接替代 & 间接替代
        - 主要看性价比，价值 = 功能 / 成本
        - 未必你死我活，也可长期共存
      3. 供应者/购买者讨价还价的能力
        - 交易对手集中程度/业务量的大小
        - 产品差异化程度与资产专用性程度
        - 纵向一体化程度
        - 信息掌握的程度(比如交易对手的转换成本)
        - 交易对手处于价值链的不同环节
        - 劳动者组织起来的程度
      4. 产业内现有企业的竞争：导致产业内竞争加剧的因素(x5)
        - 竞争对手数量众多/势均力敌
        - 产业发展缓慢
        - 顾客认为所有的商品都是同质的
        - 存在过剩生产能力
        - 进入障碍低而退出障碍高
    + 五力模型的局限性：
      1. 静态的
      2. 不适用非营利机构
      3. 假设过于理想
      4. 假设分析者有上帝视角
      5. 低估交易对手方合作的可能性
      6. 对产业竞争力构成要素考虑不全面：<大卫·亚非>互补的程度；控制互补品、捆绑式经营、交叉补贴销售
  3. 成功关键因素(KSF)分析：*不是很重要*
    + 随着产业的不同而不同；同一产业中也会因驱动因素和竞争环境的变化而变化
    + 随着产品生命周期的演变而发生变化
    + 同一产业中的各个企业会有不同的侧重
    + KSF是产业和市场层次的特征
- 竞争环境分析：侧重直接竞争对手；竞争对手、战略群组
  1. 竞争对手分析：
    + 竞争对手的未来目标
      1. 竞争对手目标分析对本公司制定竞争战略的作用：寻找既能达到目的又不威胁对手的位置
      2. 竞争对手业务单位目标分析
      3. 母公司对其业务单位未来目标的影响分析
    + 竞争对手的假设(预测)
    + 竞争对手的现行战略
    + 竞争对手的能力
      1. 核心能力
      2. 成长能力
      3. 快速反应能力：对其他公司行动的反应
      4. 适应变化的能力：对环境/外部事件的反应(更加笼统和宏观)
      5. 持久力
  2. 战略群组分析：*2015年考过了*
    + 定义：某一个产业中在某一战略方面采用相同/相似战略，或具有相同战略特征的公司组成的集团
    + 战略群组分析的作用(x4)：*背诵*
      1. 有助于很好地了解战略群组间的竞争状况
      2. 有助于了解各战略群组之间的移动障碍
      3. 有助于了解战略群组内企业竞争的主要着眼点
      4. 可以用于预测市场变化或发现战略机会

### 第二节 企业内部环境分析
- 企业资源与能力分析：记忆例子
  1. 企业资源分析
    + 企业资源的主要类型(x3)：
      1. 有形资源：物质资源&财务资源<香港半岛酒店>
      2. 无形资源：<商誉/技术专利/商标/企业文化>
      3. 人力资源
    + 决定企业竞争优势的企业资源判断标准(x4)：一个具体的资源可能满足多个标准
      1. 资源的稀缺性
      2. 资源的不可模仿性
        - 物理上独特的资源
        - 具有路径依赖性的资源<海尔的售后服务好>
        - 具有因果含糊性的资源<美国西南航空公司的企业文化>
        - 具有经济制约性的资源
      3. 资源的不可替代性
      4. 资源的持久性
  2. 企业能力主要类型：
    + 研发能力
    + 生产管理能力
    + 营销能力
    + 财务能力
    + 组织管理能力
  3. 企业的核心能力
    + 概念：比竞争对手做得更好的能力；资源 + 能力 = 核心能力
    + 辨别标准(x3)：同时满足
      1. 对顾客有价值
      2. 与竞争对手相比有优势
      3. 难以模仿或复制
    + 核心能力评价方法(x5)：
      1. 企业自我评价
      2. 产业内部比较
      3. 基准分析：与最一流的企业做比较
        - 基准对象：
          1. 占用较多资金的
          2. 能显著改善与顾客关系的
          3. 能最终影响企业结果的
        - 基准类型(x5)：
          1. 内部基准
          2. 竞争性基准
          3. 过程或活动基准：具有类似核心职能领域，多为不同产业
          4. 一般基准：有相同业务功能的企业，多指同一产业而无竞争关系的企业
          5. 顾客基准
      4. 成本驱动力和作业成本法
      5. 搜集竞争对手的信息
    + 企业核心能力与成功关键因素辨析
      1. 成功关键因素(KSF)是产业和市场层次的特征，并不针对具体公司
      2. 拥有KSF是获得竞争优势的必要条件而非充分条件
      3. 核心能力和KSF的共同之处在于他们都是公司盈利能力的指示器
- **价值链分析**：
  1. **价值链的两类活动**：
    + 基本活动(x5)：
      1. 内部后勤：进货物流，有形物
      2. 生产经营
      3. 外部后勤：出货物流
      4. 市场销售：4Ps，广告、定价等
      5. 服务：售后服务，培训客户、修理、调试等
    + 支持活动(x4)：
      1. 采购管理：广义资源投入的管理，签约谈判、聘请咨询公司等；是职能，无形的
      2. 技术开发：广义概念，生产/非生产性技术
      3. 人力资源管理
      4. 公司基础设施：**管理领域的** 基础设施(组织结构/控制系统/企业文化)，不是生产用的产房设备
  2. 企业资源能力价值链分析的使用要点(x3)：
    + [单个活动] 确认支持企业竞争优势的关键性活动
    + [企业内部联系] 明确价值链内各种活动之间的联系
    + [企业外部联系] 明确价值系统内各项价值活动之间的联系：企业与企业的价值链之间的联系
- **业务组合分析**：波士顿矩阵、通用矩阵
  1. **波士顿矩阵**：*2015年考过了* <2x2矩阵>
    + 两个维度：
      1. 纵轴：外部特点，市场增长率；企业所在产业(总体)年销售额增长率 [(本期-上期)/上期]；10%为界限
      2. 横轴：内部特征，相对市场占有率；企业某项业务的市场份额 / 最大竞争对手的市场份额；1.0为界限
    + 四种业务：结合产品生命周期，问题业务为导入期+逆时针
      1. 高增长+高竞争：明星业务
        + 高收入+高消耗；应优先供给所需资源
        + 适宜战略：扩大市场规模，提高占有率，加强竞争地位，为了最终成为现金牛而努力
        + 管理：采用事业部形式；负责人技术+销售
      2. 高增长+低竞争：问题业务
        + 低收入+高消耗：现金流情况差；应选择性投资
        + 适宜战略：判断业务前景，对有希望的进行扶持，为了成为明星而努力
        + 管理：智囊团/项目组织(有灵活性的，如矩阵式组织结构)；负责人有规划能力+敢冒风险
      3. 低增长+高竞争：现金牛业务
        + 高收入+低消耗：成熟的低速增长市场，现金流充裕；支持其他业务发展
        + 适宜战略：收获战略，压缩投资+榨取利润
        + 管理：事业部制，负责人市场营销
      4. 低增长+低竞争：瘦狗业务
        + 低收入+低消耗：前景差
        + 适宜战略：撤退战略，减小批量，缩小经营范围/直接淘汰；将企业资源向其他产品转移
        + 管理：与其他事业部合并管理
    + 四种战略目标：
      1. 发展：问题、明星
      2. 保持：现金牛(较大的)
      3. 收割：问题、(处境不佳的)现金牛、瘦狗
      4. 放弃：问题、瘦狗
    + 局限性：
      1. 数据确定较为困难
      2. 过于简单：指标单一，划分过粗
      3. 暗含市场份额与投资回报成正比的假设，与现实不符
      4. 假设资金是主要资源
      5. 有具体运用的困难
  2. 通用矩阵：行业吸引力矩阵；波士顿矩阵的推广 <3x3矩阵>
    + 两个维度，均增加了中间等级并使用了综合指标：
      1. 纵轴：产业吸引力
      2. 横轴：企业竞争地位
    + 圆圈和阴影：
      1. 圆圈面积与产业总体规模成正比
      2. 圆中扇形阴影表示该业务市场占有率
    + 应用：
      1. 左上方三个方格(吸引力高/竞争地位强)的业务适宜采取增长/发展战略，优先分配资源
      2. 右下方三个方格(吸引力低/竞争地位弱)的业务适宜采取停止/转移/撤退战略
      3. 对角线三个方格适宜采取维持/选择性发展战略
    + 局限：
      1. 综合指标在不同产业适用性不一致；权数分配有难度
      2. 划分较细，使用上较繁杂

### 第三节 SWOT分析
- 基本原理：内外部结合；内部优劣指企业特有的，相对竞争对手而言的；外部主要指环境，是产业内企业共享的
  1. S 企业内部的优势(Strengths)
  2. W 企业内部的劣势(Weakness)
  3. O 企业外部环境的机会(Opportunities)
  4. T 企业外部环境的威胁(Threats)
- **应用**：内外组合 <2x2矩阵>
  1. SO：增长型战略
  2. ST：多元化战略
  3. WO：扭转型战略
  4. WT：防御型战略

## 第三章 战略选择
章节很长也很重要

### 第一节 总体战略(公司层战略)
- 发展战略
  1. 一体化战略
    + 纵向一体化：“坐北朝南”判断前后向
      1. 优点：
        - 节约交易成本
        - 控制稀缺资源
        - 保证关键投入质量
        - 获得新客户
      2. 缺点：增加内部管理成本
      3. 分类：前向/后向一体化
        - 前向一体化适用条件(x4)：向下游；控制销售过程和渠道，增强对消费者需求变化的敏感性
          1. 现有销售商销售成本较高/可靠性较差，难以满足需要
          2. 所在产业增长潜力较大
          3. 具备一体化所需的资金和人力资源等
          4. 销售环节的利润率较高
        - 后向一体化适用条件(x6, 4+2)：向上游；控制供应商、原材料成本/质量/可靠性；汽车/钢铁行业较多采用
          1. 现有供应商供应成本较高/可靠性较差，难以满足需求
          2. (\*)供应商数量较少而买家众多
          3. 所在产业增长潜力较大
          4. 具备一体化所需资金和人力资源等
          5. 供应环节利润率较高
          6. (\*)产品价格稳定对企业十分关键，后向一体化有利于控制原材料成本
      4. 纵向一体化的风险(x2)：
        - 不熟悉新领域的风险
        - 后向一体化涉及的投资数额较大/资产专用性较强，增加企业退出成本
    + 横向一体化适用情形(x5)：收购/兼并/联合同一战略群组的直接竞争对手
      1. 产业竞争激烈
      2. 产业规模经济较显著
      3. 横向一体化符合反垄断法律法规，能在局部地区获得一定垄断地位
      4. 产业增长潜力较大
      5. 具备一体化所需资金和人力资源等
  2. **密集型战略**：充分利用现有产品or服务的潜力强化竞争地位的战略
    + <安索夫> 产品-市场战略组合 <2x2矩阵>
      1. 现有市场+现有产品：市场渗透
      2. 现有市场+新产品：产品开发
      3. 新市场+现有产品：市场开发
      4. 新市场+新产品：多元化；下一部分讲
    + 市场渗透战略：扩大现有产品的市场份额
      1. 可用方法(x3)：
        - 扩大市场份额：针对整体在成长的市场
        - 开发小众市场：企业规模较小时
        - 保持市场份额：市场衰退时
      2. 适用条件(x5)：
        - 整个市场在增长
        - 为了力保销售额不下降
        - 其他企业离开现有市场
        - 拥有强大市场地位，有独特竞争优势
        - 风险低/高级管理者参与度较高/投资较低时
    + 市场开发战略：现有产品打入新市场
      1. 主要途径：
        - 开辟其他区域市场
        - 开辟细分市场
      2. 采用市场开发战略的原因(x3)：*背诵*
        - 现有产品难以转型生产新产品
        - 市场开发与产品开发结合在一起
        - 现有市场或细分市场已经饱和
      3. 适用情况(x6)：
        - 存在未开发/未饱和市场
        - 可得到新/可靠/经济/高质量的销售渠道
        - 在现有经营领域十分成功
        - 拥有扩大经营所需资金和人力
        - 存在过剩的生产力
        - 主业属于正在迅速全球化的产业
    + 产品开发战略：在原有市场上开发新产品
      1. 优点(x5)：*背诵*
        - 延长产品寿命周期
        - 提高产品差异化程度，满足市场新需求
        - 有效帮助企业发展，改善企业竞争地位
        - 有利于利用现有产品的声誉和商标吸引用户购买新产品
        - 产品开发针对性较强，较易成功
      2. 采用产品开发战略的原因：*背诵*
        - 充分利用企业对市场的了解
        - 保持领先地位
        - 从现有产品组合的不足中寻求新的机会
        - 使企业继续在现有市场中保持安全地位
      3. 适用情况(x5)：
        - 具有较高市场信誉度和顾客满意度
        - 属于适宜创新的高速发展的高新技术产业
        - 产业处于高速增长阶段
        - 企业研发能力较强
        - 主要竞争对手以类似价格提供更高质量的产品
  3. 多元化战略：新产品+新市场
    + 采用多元化战略的原因(x3)：
      1. 在现有产品或市场中持续经营并不能达到目标
      2. 留存资金超过当下扩张所需资金
      3. 与在现有产品或市场中的扩张相比，多元化利润更高
    + 分类：
      1. 相关多元化
      2. 非相关多元化
    + 企业采用多元化战略的优点(x7)
    + 实施多元化战略的风险(x5)：*背诵*
      1. 来自原有经营产业的风险
      2. 市场整体风险
      3. 产业进入风险
      4. 产业退出风险：对新进入产业的
      5. 内部经营整合风险
- 稳定战略：
  1. 适用情况
  2. 优点(x4)
  3. 风险/缺点(x2)
- 收缩战略：
  1. 采用收缩战略的原因：主动/被动
    + 主动原因(x2)
    + 被动原因(x2)
  2. 收缩战略的方式(x3)：
    + 紧缩与集中战略：关注短期效益和立竿见影的效果
    + 转向战略
    + 放弃战略：涉及产权变更，就是卖掉
  3. 实施上的困难：
    + 对企业/业务状况的判断：何时收缩
    + 退出障碍(x5)：付出的代价
- 发展战略的主要途径：
  1. 外部发展(并购)：取得外部资源
    + 并购类型(x4)
    + 并购动机(x3)：*背诵*
      1. 避开进入壁垒，迅速进入争取市场机会，规避各种风险
      2. 获得协同效应
      3. 克服企业负外部性，减少竞争，增强对市场的控制力
    + 并购失败原因(x4)：*背诵*
      1. 决策不当的并购
      2. 并购后不能很好地进行企业整合(文化融合)
      3. 支付过高的并购费用
      4. 跨国并购面临政治风险
  2. 内部发展(新建)：利用自身资源
    + 动因(x9)
    + 缺点(x5)
    + 应用条件(x3)
    + 克服进入障碍能力的表现(x3)
  3. **战略联盟**(折中)：横向/纵向
    + 基本特征(x3)：
      1. 经济组织形式：企业与市场的中间组织
      2. 企业关系：平等的合作伙伴关系
        - 相互往来的平等性
        - 合作关系的长期性
        - 整体利益的互补性
        - 组织形式的开放性
      3. 企业行为：战略性合作行为
    + 企业战略联盟形成的动因(x6)：*背诵*
      1. 促进技术创新
      2. 避免经营风险
      3. 避免或减少竞争
      4. 实现资源互补
      5. 开拓新的市场
      6. 降低协调成本
    + 主要类型(x3)：股权参与+契约联结
      1. 合资企业：最常见；股权参与
      2. 相互持股投资：股权参与；仅持有少量股份
      3. 功能性协议：契约联结；常见形式(x4)
    + 股权式和契约式主要区别(x5)：
      1. 需要法人地位的经济实体与否
      2. 有无主次之分
      3. 利益分配方式
      4. 初始投入大小
      5. 信任感与灵活性
    + 战略联盟的管控(x2)
